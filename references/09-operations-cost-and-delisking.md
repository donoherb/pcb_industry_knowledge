# 运营、成本与去风险

## 定位

本文件补充PCB与IC载板项目中经常决定成败、但在行业综述里容易写浅的部分：

- 工厂运营的真实观察指标。
- 成本、报价和利润的拆解方式。
- 新项目导入、扩产和供应链切换的去风险逻辑。

它既适用于公司研究，也适用于内部建厂、技改、项目可研和客户导入评审。

## 工厂运营的基本观察框架

单看产能和收入，通常不足以判断一家PCB/载板工厂的经营质量。更有效的是同时看六组指标：

| 维度 | 典型指标 | 管理含义 |
| --- | --- | --- |
| 接单 | 开案数、报价转化率、样品转量产率、订单能见度 | 看需求质量而非只看订单总额 |
| 计划 | 投料达成率、排产稳定性、插单比例、急单比例 | 看计划是否被客户波动不断打乱 |
| 制造 | `FPY`、工序报废、返工率、瓶颈稼动率、`OEE` | 看工艺成熟度和设备效率 |
| 质量 | 客诉PPM、内部逃逸、过杀、8D闭环周期 | 看质量系统是否前馈而非救火 |
| 交付 | `OTD`、周期天数、在制品周转、库存周转 | 看现金和客户体验 |
| 财务 | 单位面积成本、单位增值、人均产出、现金回收周期 | 看规模是否转成经营回报 |

## 产能不是单一数字

PCB/载板行业常见的“产能”至少有五种口径：

| 口径 | 含义 | 常见误区 |
| --- | --- | --- |
| 设计产能 | 理想配置下的理论产出 | 容易高估，忽略产品结构 |
| 安装产能 | 设备装完后的名义能力 | 不等于可生产合格品 |
| 试生产产能 | 已能跑小批量 | 不等于量产节拍稳定 |
| 有效产能 | 考虑工艺窗口、维护和良率后的产出 | 更接近经营实际 |
| 合格产出产能 | 最终可交付给客户的合格量 | 对财务最重要，但公开最少 |

尤其对HDI、mSAP、ABF载板和大尺寸高速板，产品结构变化会显著改变同一条线的真实产能。

## 瓶颈管理

工厂经营问题经常不是“整体都不行”，而是局部瓶颈拉低系统输出。

### 识别瓶颈时应同时看

- 设备节拍和换线时间。
- 良率损失是否集中在该工序。
- 返工、复判、实验室验证是否占用隐性能力。
- 上下游工序是否因等待或堆积造成二次浪费。

### 常见隐形瓶颈

| 隐形瓶颈 | 典型表现 |
| --- | --- |
| 工程资料释放慢 | 设备空着，但订单不能投 |
| 实验室切片/TDR/XRF吞吐不足 | 生产做好了，检验放行跟不上 |
| AOI/AVI复判能力不足 | 误判多、良品滞留、返工决策慢 |
| 程序切换和首件确认慢 | 多品种小批量时产能被吃掉 |
| 维护保养后恢复时间长 | 稼动率数据看似正常，实际输出不稳 |
| 外协节拍不可控 | 主流程稳定，但某工序交期拖慢整体周期 |

## 制造周期 `Cycle Time` 的拆法

交付慢不一定是设备慢，也可能是等待和决策慢。

制造周期可拆为：

- 等待工程资料时间。
- 等待排产和投料时间。
- 实际加工时间。
- 等待检验/复判时间。
- 等待返工或异常处置时间。
- 等待包装和出货时间。

缩短周期最有效的动作通常不是一味加设备，而是减少异常停滞、版本切换和复判等待。

## 成本结构

PCB/载板成本分析至少应拆成以下几类：

| 成本项 | 典型内容 | 经营含义 |
| --- | --- | --- |
| 材料 | CCL、PP、铜箔、ABF/BT、药水、干膜、表面处理材料 | 高端产品中材料认证和损耗都关键 |
| 直接人工 | 操作、检验、工程支持 | 多品种和高复杂度产品人工弹性更弱 |
| 制造费用 | 水电、环保、气体、治具、维修 | 环保和洁净度会显著影响高端产线 |
| 折旧摊销 | 设备、厂房、实验室 | 新线爬坡时压力最大 |
| 质量成本 | 报废、返工、复测、客诉、索赔 | 很多企业低估这一项 |
| 管理和研发 | 项目管理、工程试验、客户导入 | 高端板和载板不能简单视作期间费用浪费 |

### 单位成本的常见拆法

- 按面积：适合标准PCB、高多层和高速板对比。
- 按panel或strip：适合工厂排产和产能分析。
- 按单颗/单package：更接近封装客户端感知，但需谨慎换算。
- 按良品成本：比“投入成本”更能反映真实盈利质量。

## 报价逻辑

健康报价不是“材料成本加成”，而是至少包含：

1. 标准材料和特殊材料差异。
2. 工艺复杂度溢价。
3. 良率风险补偿。
4. 急单和小批量切换成本。
5. 认证、打样和工程支持成本回收。
6. 客户年度降价、返利、索赔和付款条款影响。

### 报价时容易漏掉的成本

- 首件、样品和验证批的额外损耗。
- 复杂产品的切片、阻抗、可靠性和FA成本。
- 小批量高频换线造成的设备占用。
- 客户指定材料的最低采购量和呆滞风险。
- 包装、防潮、防静电和特殊物流要求。

## 新项目导入 `NPI`

NPI的核心不是“按计划过节点”，而是尽早暴露不成立假设。

建议按四个阶段判断：

| 阶段 | 主要目标 | 放行条件 |
| --- | --- | --- |
| 立项 | 图纸、应用、标准和商业目标清楚 | 风险清单、责任人、资源和时间冻结 |
| 样品 | 证明可制造和关键性能成立 | 关键尺寸、电性、外观和初步可靠性通过 |
| 小批量 | 证明过程可重复 | `FPY`、节拍、变差和追溯达到预设门槛 |
| 量产 | 证明可持续交付和盈利 | 客户放行、稳定能力、异常反应和成本闭环 |

如果样品阶段靠工程师手工补救才能通过，通常说明量产风险尚未关闭。

## Safe Launch / 受控量产

对汽车板、高速板、ABF/BT载板、新客户导入项目，受控量产比“直接转正常量产”更稳妥。

受控量产阶段通常加强：

- 更高抽样频率。
- 首几批全检或附加验证。
- 专人复判和客户日报机制。
- 关键材料和关键设备锁定。
- 异常升级和停线权限前置。

受控量产不是形式动作，其目的在于用较低成本吸收早期波动，避免问题直接流向客户端。

## 扩产去风险

扩产失败通常不是因为“行业不增长”，而是扩产路径与客户、良率和现金流脱节。

### 扩产项目最小核对项

1. 目标产品和目标客户是否具体。
2. 名义产能是否已换算为有效产能和良品产出。
3. 瓶颈设备、实验室、环保和洁净度是否同步配置。
4. 关键工程人才、班组长和维护能力是否先于设备到位。
5. 认证节奏、送样节奏和折旧启动时点是否匹配。
6. 良率爬坡预算是否覆盖报废、返工和客户验证延迟。
7. 若客户导入延迟，现金流是否仍可承受。

### 扩产常见失败模式

| 失败模式 | 典型根因 |
| --- | --- |
| 设备已到位但产出上不来 | 工艺窗口、人才、程序和实验室没跟上 |
| 有收入但不赚钱 | 折旧、良率和小批量验证成本吞噬毛利 |
| 能做低阶版本，高阶订单迟迟进不来 | 客户认证和封装/装配协同不足 |
| 产线忙但现金紧 | 急单、小批量和高库存占用营运资金 |
| 名义稼动率高但客户不满意 | 交付结构差、返工多、稳定性不足 |

## 供应商质量与切换管理

材料和外协切换对PCB/载板的风险，常被低估为“采购替代”问题。实际更像“工艺窗口重建”。

切换时至少应确认：

- 是否需要客户批准或再认证。
- 新旧供应商的关键参数是否真的可替换。
- 药水、干膜、铜箔、ABF/BT切换后是否重做可靠性和SPC基线。
- 安全库存和双供策略是否覆盖切换窗口。
- 外协厂是否使用同版本标准和追溯规则。

## 质量成本 `COPQ`

很多板厂利润不佳，不是因为报价低，而是被隐形 `COPQ` 吃掉。

`COPQ` 常见构成：

- 报废。
- 返工和复测。
- 加急补料和重排产。
- 客诉退货、索赔和驻厂。
- 工程和质量团队长期救火的隐性人力。

判断一家工厂是否真的改善，不能只看出货量，还要看 `COPQ` 是否下降、异常是否前移、8D是否减少重复问题。

设备维护、工装治具、OEE、备件和设备变更会直接影响瓶颈产能、良率和交付稳定性，详见 `34-equipment-maintenance-and-tooling.md`。EHS、废水废气、危废和化学品管理会影响扩产边界、成本和连续生产能力，详见 `35-environment-ehs-chemical-management.md`。

## 现金流与营运资金

PCB/载板项目的利润和现金流经常不同步。重点看三件事：

- 原材料和在制品占用是否随着复杂度上升而明显增加。
- 大客户账期和返利条款是否抵消了高ASP好处。
- 新项目样品、小批量和认证批是否形成大量难回收库存。

营运资金压力常在以下情形放大：

- 客户指定材料且备货周期长。
- 载板或高阶板在制周期长、批量小。
- 导入期频繁改版，旧料和旧程序报废高。
- 海外客户账期长且结算币种波动。

## 尽调与内部评审问题清单

### 对工厂运营团队

- 当前最常卡住的瓶颈工序和真正原因是什么。
- `FPY`、返工率和最终良率分别是多少，趋势如何。
- 周期最长的是哪三类产品，为什么。
- AOI/AVI复判、切片、阻抗和可靠性实验室是否跟得上。
- 过去12个月重复发生最多的三类问题是什么。

### 对财务和经营团队

- 单位良品成本如何计算，是否包含质量成本。
- 新线折旧何时开始，满产假设基于什么证据。
- 高端产品ASP提升是否被材料、报废和验证成本抵消。
- 扩产和客户导入若延后一个季度，现金流还能否承受。

### 对项目和质量团队

- 样品通过是靠系统能力还是靠人工挑选和补救。
- 控制计划、PFMEA、MSA、SPC是否围绕同一组关键特性。
- 失效分析结果是否能回写到设计和制程，而不是停留在结案报告。

## 运营判断的结论模板

对PCB/载板企业或项目，可用以下结构做简洁判断：

1. 需求质量：订单来自景气、份额还是一次性导入。
2. 制造质量：瓶颈、`FPY`、追溯和异常闭环是否成熟。
3. 交付质量：`OTD`、周期和客户端稳定性是否可验证。
4. 盈利质量：ASP能否覆盖复杂度、折旧和质量成本。
5. 扩张质量：扩产是否建立在客户、认证和现金流闭环上。

如果以上五项中只有“需求质量”成立，而其余几项没有同步成立，扩张往往会先带来收入，再带来更大的组织和现金流压力。
