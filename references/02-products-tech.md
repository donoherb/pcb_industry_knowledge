# 产品分类与技术路线

## PCB产品

| 产品 | 定义 | 关键能力 | 常见应用 |
| --- | --- | --- | --- |
| 单双面板 | 一层或两层导电图形 | 成本、交期、基础良率 | 家电、低端电源、简单控制板 |
| 多层板 | 三层及以上，通过压合形成多层互连 | 层压、钻孔、对位、阻抗 | 通信、服务器、汽车、工业 |
| 高多层板 | 层数较高、结构复杂的多层板 | 高纵横比孔、层间对准、可靠性 | 交换机、服务器、背板 |
| HDI | 使用微盲孔实现高密度互连 | 激光钻孔、填孔电镀、叠孔、mSAP | 手机、可穿戴、模组、车载 |
| FPC | 使用柔性基材的可弯折线路板 | PI材料、弯折寿命、贴装和补强 | 手机、摄像头、显示、汽车 |
| 软硬结合板 | 刚性区和柔性区结合 | 压合、开窗、可靠性、装配 | 高可靠小型化模组 |
| 高频高速板 | 面向高速信号或射频传输 | Dk/Df、阻抗、损耗、表面粗糙度 | 5G、交换机、雷达、服务器 |
| 金属基/厚铜板 | 强调散热或大电流能力 | 热阻、铜厚、绝缘层可靠性 | LED、电源、新能源车 |
| SLP | 类载板PCB，介于HDI和IC载板之间 | 细线宽线距、mSAP、尺寸控制 | 高端智能手机等小型化终端 |

## IC载板产品

IC载板按材料和封装路线常见分为BT载板和ABF载板。BT载板常用于存储、射频、SiP、部分CSP/BGA等场景；ABF载板常用于高性能处理器、GPU、CPU、AI加速器、网络芯片等更大尺寸和更高I/O需求的封装。

| 产品/路线 | 典型特征 | 关注重点 |
| --- | --- | --- |
| WB基板 | 线键合封装使用 | 成本、稳定性、成熟产能 |
| FC-CSP | 倒装芯片、尺寸较小 | 细线路、凸点匹配、翘曲 |
| FC-BGA | 倒装BGA，常见于高性能芯片 | 大尺寸、ABF材料、多层、良率 |
| SiP基板 | 系统级封装，多芯片/器件集成 | 设计协同、贴装、可靠性 |
| 存储载板 | DRAM/NAND等存储封装 | 薄型化、批量稳定性、成本 |
| RF/模组基板 | 射频和小型模块 | 高频特性、尺寸、材料一致性 |

## 关键技术指标

| 指标 | 含义 | 为什么重要 |
| --- | --- | --- |
| 层数 | 导电层数量 | 决定布线密度、信号完整性和成本 |
| 线宽线距 | 导线宽度和导线间距 | 代表图形精度和密度能力 |
| 孔径 | 机械孔或激光孔大小 | 影响互连密度、可靠性和成本 |
| 纵横比 | 板厚与孔径之比 | 影响孔金属化和可靠性 |
| 盲孔/埋孔/通孔 | 不同层间互连方式 | 影响HDI结构和布线自由度 |
| 叠孔/错孔 | 微孔堆叠或错位 | 影响密度、良率和可靠性 |
| 阻抗控制 | 控制信号路径阻抗 | 高速信号完整性关键 |
| Dk/Df | 介电常数/介质损耗 | 高频高速板材料核心指标 |
| CTE | 热膨胀系数 | 影响热循环、翘曲和封装可靠性 |
| Tg | 玻璃化转变温度 | 影响耐热和尺寸稳定 |
| 翘曲 | 板或载板平整度变化 | 影响封装、贴装和可靠性 |
| CAF | 导电阳极丝失效风险 | 高可靠板长期绝缘风险 |

## 工艺路线

减成法是传统PCB主流方法，通过铜箔蚀刻形成线路。适合成熟线宽线距和成本敏感产品。

mSAP和SAP通过半加成思路形成更细线路，是HDI、SLP和IC载板细线路能力的重要路线。判断企业是否真正具备能力，应看量产线宽线距、良率、客户认证和订单规模，而不是单片样品能力。

## PCB与IC载板的差异

| 维度 | PCB | IC载板 |
| --- | --- | --- |
| 位置 | 电子整机或模组内部 | 芯片封装内部，连接裸芯片与PCB |
| 客户 | 整机厂、EMS、模组厂、系统厂 | 封测厂、IDM、Fabless供应链 |
| 精度 | 从低到高跨度大 | 整体精度要求更高 |
| 认证 | 依应用而定 | 认证周期通常更长 |
| 成本结构 | 材料、人工、折旧、良率 | 材料、折旧、良率、洁净和认证压力更重 |
| 风险 | 周期、价格、环保、客户转单 | 良率、翘曲、客户导入、扩产爬坡 |

## 技术升级方向

- 多层和高多层：服务高速信号、电源完整性和复杂系统互连。
- 高频高速：低损耗材料、低粗糙铜箔、精密阻抗控制和背钻等能力更重要。
- HDI升级：从一阶/二阶向任意层、叠孔、填孔和更细线宽线距发展。
- FPC升级：向多层、软硬结合、高可靠汽车和小型模组扩展。
- IC载板升级：ABF大尺寸、多层、细线路、低翘曲和先进封装协同。
- 制造数字化：提升良率追溯、排产、质量闭环和能耗管理。

## 判断问题

分析某产品或公司时，至少回答：

- 产品属于哪一类，技术门槛来自材料、设备、工艺还是客户认证。
- 该技术能力是样品、小批量还是稳定量产。
- 能力指标是否有公开证据，是否来自客户认证或量产收入。
- 竞争对手是谁，替代产品或路线是什么。
- 升级是否提高ASP和毛利，还是只提高资本开支和良率风险。

## 设计规则与制造能力对应

很多技术讨论停留在图纸指标，但制造难度真正来自“设计规则如何穿透到工艺窗口”。

| 设计或产品要求 | 制造端被放大的难点 | 常见经营含义 |
| --- | --- | --- |
| 更细线宽线距 | LDI解析度、洁净度、图形电镀、AOI分辨率 | 初期良率更敏感，缺陷密度管理更关键 |
| 更高层数和更大纵横比 | 压合涨缩、钻孔、孔铜可靠性、层间对位 | 周期变长，过程能力和切片验证压力上升 |
| 低损耗高速材料 | 材料批次控制、粗糙度、阻抗和损耗测试 | 材料成本高，客户认证和电测/射频验证更重 |
| 叠孔、via-in-pad、填孔 | 微孔清洁、填孔电镀、平整度和热循环可靠性 | 报废成本高，需靠过程控制而非终检兜底 |
| 大尺寸ABF载板/薄型板 | 翘曲、共面性、吸湿和洁净度 | 新产线爬坡更慢，封装客户端判定更严 |

因此，判断技术能力时，应同时问“图纸能不能做”和“量产时良率、节拍、验证和追溯能不能跟上”。

## 先进封装和系统需求对载板的牵引

IC载板的升级不能脱离封装路线和系统需求理解。常见牵引关系如下：

- 更高I/O和更大封装体：推动更细线路、更多build-up层、更严格翘曲控制。
- 更高功耗和更高频率：推动电源完整性、热管理、低损耗材料和表面粗糙度控制。
- Chiplet和异构集成：提升封装协同复杂度，对基板尺寸稳定、共面性和良率提出更高要求。
- 封装微缩和模组集成：推动BT、ABF、SiP等不同路线分化，不能用单一“高端/低端”概括。

实践中要分清三个层次：

- 有机IC载板：连接芯片封装和PCB。
- 重新布线层或中介层：更靠近先进封装和晶圆级工艺。
- 类载板PCB/SLP：仍属于PCB体系，但向细线路和高密度互连靠近。

这三类产品在客户链、设备、良率逻辑和资本开支强度上并不相同。
