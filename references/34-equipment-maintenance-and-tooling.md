# 设备维护、工装治具与OEE

来源更新：2026-04-17。本文沉淀PCB/载板行业设备维护、工装治具、备件、预防维护、OEE和设备变更的稳定知识，不替代设备供应商手册、安全操作规程、校准要求、特种设备法规、企业EAM/TPM程序和客户批准规则。涉及生产地点、设备型号、关键工装、检测设备或程序变化时，应按客户CSR、ECN和再PPAP规则评估。

## 一句话理解

设备能力不是采购清单，而是设备状态、工装治具、程序、维护、备件、人员和过程数据共同形成的可重复制造能力。OEE高但良率差、维护弱或治具失控，不能说明产线成熟。

## 适用范围

本专题覆盖：

- 生产设备：钻孔、激光钻、压合、LDI、显影蚀刻、电镀、阻焊、表面处理、成型、电测、AOI/AVI和自动化物流。
- 检测设备：TDR、XRF、CMM、切片制样、离子污染、warpage、可靠性设备。
- 工装治具：钻孔垫板/盖板、夹具、模具、测试治具、电测针床、飞针程序、AOI/AVI程序、包装载具。
- 维护体系：点检、PM、TPM、故障维修、备件、校准、设备能力和OEE改善。
- 设备变更：搬迁、替换、大修、软件升级、夹治具改版和参数窗口变化。

## 设备生命周期

| 阶段 | 关键动作 |
| --- | --- |
| 选型 | 根据产品能力、精度、节拍、洁净度、维护资源和客户要求选型 |
| 安装调试 | 安装验证、IQ/OQ/PQ或等效验证、设备安全和基础参数确认 |
| 试产 | 用目标产品验证节拍、良率、稳定性、维护和人员能力 |
| 量产 | 点检、PM、SPC、OEE、故障分析、备件和持续改善 |
| 变更 | 设备搬迁、软件升级、大修或夹治具改变时做影响评估 |
| 退役 | 停用、替代、数据备份、程序迁移和客户批准状态处理 |

设备到厂不等于产能形成。真正的有效产能要等设备、工艺、材料、人员、维护和质量数据同时稳定。

## 设备能力与过程能力

要区分三类能力：

| 能力 | 含义 | 常见误区 |
| --- | --- | --- |
| 设备规格能力 | 设备供应商标称精度、速度和功能 | 把样机参数当作量产能力 |
| 设备状态能力 | 当前安装、校准、维护后的实际能力 | 只看证书，不看漂移和故障趋势 |
| 过程能力 | 设备在材料、程序、人员和环境下稳定产出合格品的能力 | 设备能力好但过程仍可能不稳定 |

关键客户项目应以目标产品、目标材料、目标程序和量产条件验证能力。

## 点检和预防维护

PM计划应按设备风险、供应商建议、历史故障和产品要求制定。

| 项目 | 内容 |
| --- | --- |
| 日常点检 | 安全、防护、温湿度、压力、流量、温度、报警、清洁、夹具状态 |
| 周期PM | 润滑、更换耗材、校正、清洁、滤芯、传动、光学系统、药水相关部件 |
| 精度确认 | 定位、对位、孔位、厚度、线宽、镀层、阻抗或关键设备能力 |
| PM后验证 | 首件、标准片、黄金板、能力数据或关键参数复核 |
| 异常维修 | 故障记录、根因、临时措施、复机确认和影响批次评估 |

PM不能只记录“已完成”。需要确认维护后设备是否恢复到可接受状态。

## 工装治具管理

工装治具和设备一样需要生命周期管理：

- 编号、版本、适用料号、客户、设备和状态。
- 设计、制作、验收、首件验证、放行和维护。
- 使用寿命、点检项目、磨损标准、维修和报废条件。
- 存放、防潮、防尘、防变形、防混用和权限控制。
- 变更记录、客户批准状态和旧版退出。

高风险治具包括电测针床、钻孔/成型夹具、压合辅助治具、FPC补强/冲切模具、载板strip载具和客户指定包装载具。

## 耗材寿命控制

PCB/载板的很多缺陷来自耗材寿命失控：

| 耗材 | 失控风险 | 控制方式 |
| --- | --- | --- |
| 钻咀 | 孔壁粗糙、偏孔、断针、孔铜可靠性下降 | 寿命曲线、首件、断针记录、供应商批次 |
| 铣刀 | 毛刺、尺寸偏差、边缘裂纹 | 寿命、尺寸复核、外观限度样 |
| 电测探针 | 接触不良、误判、漏测 | 点检、清洁、寿命、黄金板验证 |
| 过滤芯 | 药水污染、颗粒、镀层异常 | 压差、周期更换、槽液趋势 |
| 曝光灯/激光器 | 能量漂移、解析度下降 | 能量监控、标准图形、维护记录 |
| 吸盘/夹具 | 划伤、压痕、定位漂移 | 外观点检、清洁、寿命和防混用 |

寿命规则应来自数据，而不是只按经验固定次数。

## OEE拆解

OEE由可用率、性能和良品率组成。PCB/载板工厂看OEE时要同时看：

- 可用率：故障停机、换线、PM、待料、待程序、待检验。
- 性能：节拍、速度降级、程序切换、试做和复判等待。
- 良品率：设备导致的报废、返工、复测、客户退货和隐性挑选。

常见误判：

- 只看设备运行时间，不看做出的是否是客户可接受良品。
- 将待工程资料、待实验室和待MRB排除在设备效率外。
- 高OEE靠减少PM获得，短期好看、长期故障和质量风险上升。
- 混合产品结构变化后仍用同一OEE目标比较。

## 设备故障闭环

重大或重复设备故障应按8D或类似方法处理：

1. 保护现场和产品，锁定故障前后批次。
2. 判断是否影响质量、交付、客户批准和安全。
3. 分析发生原因：机械、电气、软件、耗材、环境、操作或维护。
4. 分析流出原因：为什么点检、报警、SPC或首件没有发现。
5. 制定永久措施：PM项目、备件、传感器、程序、防错、培训或供应商改进。
6. 验证复机效果：首件、标准样、SPC、能力或连续批次数据。
7. 回写PM计划、Control Plan、PFMEA、LPA和备件策略。

## 备件和供应商服务

关键设备备件策略要结合停机损失和供应周期：

- A类备件：停机即影响瓶颈产能或关键质量，应有安全库存。
- B类备件：可短期降级运行或替代，应有供应周期和采购触发点。
- C类备件：通用低风险，可按消耗补充。

还要关注设备供应商服务能力、远程诊断、软件版本、保密权限和备件真伪。关键设备长期依赖单一海外供应商时，维护周期会直接影响真实产能。

## 设备和治具变更

以下变化通常需要正式影响评估：

- 关键设备替换、搬迁、大修、软件升级或控制系统修改。
- 夹具、模具、电测治具、AOI/AVI程序、电测程序和包装载具改版。
- 关键设备参数窗口、报警阈值、PM周期或耗材寿命规则改变。
- 检测设备、实验室方法、校准方法或判定算法变化。
- 外协设备或生产地点变化。

评估内容包括客户批准、再PPAP、首件、MSA、SPC、能力、可靠性、库存和旧版退出。

## 审核常看点

审核通常会看：

- 设备清单、关键设备识别、维护计划和实际执行记录。
- PM后是否有复机验证，而不是只签名。
- 设备故障是否评估受影响批次和客户风险。
- 工装治具版本、寿命、点检和报废是否受控。
- OEE、停机、故障和质量损失是否用于改善。
- 设备变更是否触发ECN、客户批准或再PPAP评估。
- 检测设备是否与MSA、校准和实验室相关性联动。

## 常见错误

- 把设备采购当成产能已经形成。
- PM记录完整，但没有维护后能力确认。
- 治具没有版本和寿命管理，靠现场经验识别。
- 设备故障后只维修，不追溯故障期间生产批次。
- OEE只算设备效率，不纳入质量损失和换线影响。
- 设备软件升级、程序修改没有走变更评估。
- 关键备件缺失，导致瓶颈设备长时间停机。
- 检测设备校准合格，但未做MSA或相关性验证。

## 快速落地流程

`关键设备识别 -> 设备能力验证 -> 点检/PM/备件策略 -> 工装治具编号和寿命控制 -> OEE和故障数据监控 -> 故障8D和复机验证 -> 设备/治具变更评估 -> 回写PFMEA/Control Plan/LPA/维护计划`
