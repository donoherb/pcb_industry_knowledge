## 定位

本文件聚焦汽车电子PCB，包括域控制器、BMS、OBC/DC-DC、车身控制、摄像头/雷达模组板等。汽车板的核心不是“参数更高”，而是长期可靠性、追溯、项目管理和客户体系要求更重。

## 应用分类

| 应用 | 典型板种 | 关注重点 |
| --- | --- | --- |
| 域控/中央计算 | 多层、高速、高可靠板 | 热管理、可靠性、功能安全接口 |
| BMS/电源控制 | 厚铜板、功率板 | 电流、热循环、绝缘距离 |
| OBC/DC-DC/逆变周边 | 厚铜、金属基或高可靠多层 | 散热、电流、耐压、振动 |
| 摄像头模组 | HDI/FPC/软硬结合 | 小型化、装配、光学模组协同 |
| 毫米波雷达板 | 高频/高频混压板 | Dk/Df、尺寸稳定、射频性能 |
| 车身与舒适系统 | 多层和普通高可靠板 | 成本、追溯、寿命和环境适应 |

## 汽车板的本质要求

汽车板与消费类产品的根本差别在于：

- 生命周期更长。
- 环境应力更复杂。
- 客户审核和变更管理更严格。
- 客诉代价和召回风险更高。

因此，汽车板竞争力通常来自“项目管理 + 过程质量 + 可靠性 + 追溯”，而不只是单点工艺能力。

## 项目导入逻辑

汽车项目通常遵循更长的导入周期：

客户需求/CSR -> APQP项目策划 -> 材料和设计确认 -> 样品验证 -> 小批量试生产 -> PPAP/客户批准 -> 量产 -> 变更和持续审核。

其中关键不是文件齐不齐，而是特殊特性、失效模式、测量系统、过程能力和反应计划是否真实落地。

## 车规质量关注点

| 维度 | 重点 | 常见风险 |
| --- | --- | --- |
| 可靠性 | 热循环、振动、湿热、盐雾、寿命 | 早期失效、批次波动 |
| 追溯 | lot、panel、材料、设备、班次、出货追溯 | 客诉无法界定影响范围 |
| 变更管理 | 材料、工艺、设备、外协变更受控 | 未经批准变更导致客户风险 |
| 测量系统 | CTQ项目量测可信 | 能力判断失真 |
| PFMEA/Control Plan | 特殊特性与控制点一致 | 文件与现场脱节 |

## 毫米波雷达板专项

车载雷达板既有汽车质量属性，又有射频和高频设计属性。

重点包括：

- 高频材料一致性。
- 介质厚度和铜面粗糙度控制。
- 线路和天线区域尺寸精度。
- 表面处理对高频性能和焊接的平衡。
- 模组装配后实际射频性能。

雷达板不能只按普通高速板逻辑分析，因为其天线和射频结构对几何误差更敏感。

## 功率与BMS板专项

高电流和高压场景常见于BMS、OBC/DC-DC和功率控制相关板。重点不只在铜厚，还在：

- 载流能力和温升。
- creepage / clearance 设计。
- 厚铜蚀刻与图形精度平衡。
- 热循环下焊点和铜界面可靠性。
- 涂覆、灌封和整机装配适配。

## 常见失效机理

| 机理 | 典型诱因 | 场景 |
| --- | --- | --- |
| 孔铜疲劳裂纹 | 宽温循环、厚板小孔、材料CTE失配 | 域控、电源板 |
| 分层/起泡 | 吸湿、回流、热冲击 | 多层、HDI、厚铜板 |
| `CAF` / 绝缘失效 | 湿热偏压、污染、间距不足 | 高可靠长期应用 |
| 焊接失效 | 表面处理、装配热曲线、板弯 | 模组板、电源板 |
| 高频性能漂移 | 材料波动、几何偏差 | 雷达板 |
| 振动裂纹 | 连接器/重器件机械应力 | 域控、电源、模组板 |

## 典型验证矩阵

汽车板验证一般比消费电子更重，常见组合包括：

- 热循环/热冲击。
- 振动/机械冲击。
- 湿热偏压和长期绝缘验证。
- 盐雾或腐蚀环境验证。
- 可焊性和回流后结构验证。
- 高频板的射频性能验证。

具体条件仍以客户CSR、图纸和验证计划为准，不能用知识库默认值替代。

## 制造与运营难点

| 难点 | 说明 | 经营含义 |
| --- | --- | --- |
| 多品种长生命周期 | 平台切换慢、旧料管理复杂 | 库存和变更管理压力大 |
| 文档和审核要求高 | PPAP、审核、CSR维护重 | 间接成本更高 |
| 追溯要求严 | 数据系统必须稳定 | 数字化和纪律性是门槛 |
| 客诉容忍度低 | 8D和遏制响应要快 | 质量组织能力很关键 |
| 新项目导入慢 | 样品到量产周期长 | 收入确认和扩产节奏需保守 |

## 评估汽车板企业时的红旗

- 拿汽车客户送样或小批量当成稳定量产证据。
- 只讲IATF证书，不讲具体CSR和客户批准状态。
- 过程能力、MSA和Control Plan停留在文档层。
- 客诉、召回、批次隔离和追溯能力说不清。
- 高频雷达板能力只讲材料，不讲天线尺寸控制和射频验证。

## 汽车板专项尽调清单

1. 主力汽车产品是域控、功率、BMS还是雷达板。
2. 特殊特性和关键可靠性项目由谁定义，如何下沉到现场。
3. PPAP/客户批准处于样品、小批量还是正式量产。
4. 变更管理是否覆盖材料、设备、参数、外协和程序。
5. 客诉后多久能界定影响lot和在制品范围。
6. 对雷达板，是否能把射频指标和制造公差关联起来。
7. 对功率板，是否能把铜厚、电流、温升和寿命验证闭环。
