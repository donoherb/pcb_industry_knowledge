# 材料、设备与耗材

## 材料结构

PCB和IC载板的材料决定电气、热、机械和可靠性边界。材料不是简单采购项，通常与客户认证、工艺窗口和良率绑定。

| 类别 | 典型材料 | 关键指标 |
| --- | --- | --- |
| 覆铜板 | FR-4、高Tg、无卤、高频高速CCL | Tg、Dk、Df、CTE、剥离强度、耐热 |
| 半固化片 | PP | 树脂含量、流胶、厚度均匀性、介电性能 |
| 铜箔 | 标准铜箔、低轮廓铜箔、极薄铜箔 | 厚度、粗糙度、延展性、剥离强度 |
| 树脂体系 | 环氧、BT、PTFE、PPO/PPE、PI等 | 损耗、耐热、吸水率、加工性 |
| ABF膜 | ABF buildup film | 细线路、绝缘、尺寸稳定、供应安全 |
| BT材料 | BT树脂基材料 | 耐热、机械稳定、批量一致性 |
| 柔性材料 | PI、FCCL、覆盖膜、胶系材料 | 弯折、耐热、尺寸稳定 |
| 表面处理材料 | ENIG、ENEPIG、OSP、沉银、沉锡等 | 焊接性、平整度、可靠性、成本 |
| 油墨/干膜 | 阻焊油墨、感光干膜 | 分辨率、附着力、耐化学性 |

## 设备

| 工序 | 设备 | 关注点 |
| --- | --- | --- |
| 开料/内层 | 开料、清洗、曝光、显影、蚀刻 | 图形精度、洁净度、均匀性 |
| 钻孔 | 机械钻、激光钻 | 孔径、定位、毛刺、效率、钻针消耗 |
| 压合 | 真空压机、叠合系统 | 层间对准、树脂流动、翘曲 |
| 电镀 | 沉铜、电镀铜、填孔电镀 | 均匀性、孔可靠性、药水稳定 |
| 图形转移 | 直接成像LDI、曝光机 | 解析度、对位、产能 |
| 检测 | AOI、AVI、X-ray、飞针、电测 | 缺陷识别、误判率、追溯 |
| 成型 | 铣边、冲切、激光切割 | 尺寸、毛刺、边缘可靠性 |
| 环保 | 废水、废气、回收系统 | 达标、成本、产能约束 |

IC载板和高端HDI更依赖高精度曝光、激光钻孔、精密电镀、洁净环境和自动检测。设备买到不等于量产能力，工艺调试和工程经验同样关键。

## 耗材和药水

耗材和药水影响良率和一致性，低成本替代不能只看采购单价。

- 钻针和铣刀：影响孔壁质量、尺寸精度和加工效率。
- 干膜和油墨：影响图形解析度、阻焊桥和可靠性。
- 沉铜、电镀、蚀刻药水：影响孔铜、线宽控制、均匀性和可靠性。
- 清洗和微蚀体系：影响附着力、洁净度和后续缺陷。
- 表面处理药水：影响焊接、打线、接触可靠性和保存期。

## 成本与价格传导

PCB成本通常受铜价、覆铜板、人工、能耗、环保、折旧和良率影响。低端产品更容易体现材料价格波动，高端产品更看客户结构和制程能力。

IC载板中，高端材料、设备折旧、良率和试产成本占比更高。扩产初期即使收入增长，利润也可能被折旧和良率爬坡吞噬。

## 供应链风险

- 单一材料或设备供应商依赖。
- 高端材料受限或交期长。
- 客户指定材料导致替代空间小。
- 环保和能耗限制影响产能。
- 汇率、关税和贸易限制影响进口材料设备。
- 新材料导入改变工艺窗口，造成初期良率波动。

## 尽调问题

研究材料或设备相关问题时，询问：

- 该材料是否已通过终端客户或封装客户认证。
- 对应产品收入是否已经量产，而不是处于送样阶段。
- 供应商切换需要多久，是否影响可靠性。
- 是否存在关键材料长协、价格联动或最低采购承诺。
- 新设备到厂、安装、调试、认证和满产需要多久。
- 设备瓶颈位于钻孔、曝光、电镀、压合、检测还是环保。

## 材料选型逻辑

材料选择不是单纯比参数高低，而是看“应用需求是否值得为该材料付出工艺和成本代价”。

| 场景 | 材料关注点 | 常见隐藏成本 |
| --- | --- | --- |
| 服务器/AI高速板 | Dk/Df、铜箔粗糙度、厚度一致性、吸湿 | 测试验证、阻抗调试、压合窗口收窄 |
| 汽车电子 | Tg、CTE、CAF、热循环和耐湿热 | 认证周期长，来料和可靠性验证成本高 |
| FPC动态弯折 | PI/FCCL、压延铜、胶系和覆盖膜 | 弯折寿命验证、补强和贴装协同复杂 |
| 高频射频板 | PTFE/PPO/PPE等材料体系、表面粗糙度 | 加工难度、钻孔和表面处理适配要求高 |
| ABF/BT载板 | 吸湿、尺寸稳定、洁净度、build-up匹配 | 储存、防潮、warpage和良率爬坡成本高 |

参数更高不一定更优。若客户并不为更高规格付费，过度用料会直接侵蚀毛利和制造窗口。

## 设备瓶颈识别

设备瓶颈往往决定企业真实产能上限。分析时要区分“名义瓶颈工序”和“真正限制良率与节拍的工序”。

| 产品段 | 常见瓶颈设备/环节 | 为什么是瓶颈 | 常见误判 |
| --- | --- | --- | --- |
| 高多层板 | 压合、钻孔、电镀、切片实验室 | 层数高，工序时间长且返工代价大 | 只盯钻机数量，忽略压合和验证节拍 |
| HDI | 激光钻孔、填孔电镀、LDI、AOI | 微孔和细线路缺陷密度高 | 只看样品能力，不看量产良率 |
| mSAP/SLP | 高精度曝光、图形电镀、清洁和高分辨率检测 | 线宽线距更细，颗粒和残铜影响更大 | 把设备采购等同于量产能力 |
| FPC/软硬结合 | 覆盖膜贴合、补强、冲切、贴装适配 | 柔性材料易变形，工艺窗口窄 | 忽略装配端反馈和弯折寿命 |
| IC载板 | 曝光、电镀、洁净环境、warpage量测 | 微缺陷、共面性和洁净度要求更高 | 只看制板，不看封装匹配性 |

做产能判断时，建议同时看设备台数、节拍、稼动率、良率、维修保养、程序切换时间和验证实验室吞吐，而不是只看公开的设备采购清单。
