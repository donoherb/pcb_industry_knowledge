## 定位

本文件聚焦AI服务器、交换机、路由器、加速卡和高速背板相关PCB。重点不在泛泛而谈“高速板”，而在于这类产品对材料、信号完整性、热设计、制造窗口和经营质量提出的特殊要求。

## 应用边界

典型对象包括：

- AI训练/推理服务器主板和加速卡板。
- 交换机、路由器、光模块配套高速板。
- 服务器背板、mezzanine板、OAM/OCP加速模块相关板。
- 高层数、高速互连、高功耗和高散热约束下的PCB。

这类板往往不以“小型化”为主，而以高速传输、供电完整性、散热和系统可靠性为核心。

## 需求牵引

AI服务器和交换机高速板的需求，主要由四个因素驱动：

1. 更高SerDes速率和更低信号损耗预算。
2. 更高供电电流密度和更严格的电源完整性。
3. 更高端口密度、连接器密度和布线复杂度。
4. 更大系统功耗和散热约束。

因此，这类板的价值量通常不是只靠层数，而是靠“材料 + 叠构 + SI/PI + 制造稳定性 + 客户验证”共同决定。

## 常见产品形态

| 形态 | 典型特征 | 重点关注 |
| --- | --- | --- |
| 服务器主板 | 多层、高速CPU/GPU/内存/PCIe互连 | 层数、阻抗、低损耗材料、供电平面 |
| 加速卡板 | 高速I/O、高电流、局部高热流密度 | 大铜厚、电源完整性、散热、翘曲 |
| 交换机板 | 高端口密度和高速差分对 | 损耗、串扰、连接器区域布线 |
| 背板 | 超长链路、多连接器和高可靠性 | 超低损耗、背钻、stub控制、插拔可靠性 |
| 光模块配套板 | 高频、高精度和较强EMI要求 | 高频材料、尺寸稳定、表面粗糙度 |

## 技术主线

### 1. 材料体系

高速板常用低损耗材料、低粗糙铜箔和更严格的介质厚度控制。关键不只是材料牌号，而是材料批次一致性、加工窗口和阻抗模型是否闭环。

### 2. 叠构与SI/PI

高速板设计常同时面对：

- 差分阻抗控制。
- 串扰控制。
- return path连续性。
- stub控制与背钻。
- 电源平面噪声和去耦网络。

对制造端来说，最重要的是把这些设计目标转成可量测、可放行、可追溯的制造条件。

### 3. 热设计

AI服务器和加速卡功耗高，局部热点明显。热设计常与PCB本体深度耦合，包括铜厚、导热路径、热通孔、散热器接口和板弯控制。

### 4. 机械与装配

大型板件、高连接器密度和散热件重量，使这类产品对板厚、刚性、弯曲和插拔机械可靠性更敏感。

## 关键设计与制造指标

| 指标 | 重要性 | 常见风险 |
| --- | --- | --- |
| 低损耗 `Dk/Df` | 决定通道损耗预算 | 材料批差导致链路余量不足 |
| 阻抗窗口 | 决定SI可实现性 | 线宽/介质厚度漂移导致失配 |
| 表面粗糙度 | 影响高频损耗 | 铜面粗糙使插损恶化 |
| 背钻质量 | 降低stub影响 | 残stub过大引起反射 |
| 层间对位 | 决定连接器、过孔和参考层一致性 | 对位漂移影响通道性能 |
| 板厚和平整度 | 影响连接器装配和机械稳定 | 插拔应力和装配失配 |
| 铜厚与电流能力 | 影响供电和温升 | 局部发热、压降过大 |

## 工艺难点

| 难点 | 制造挑战 | 经营含义 |
| --- | --- | --- |
| 高层数 | 压合次数多、涨缩控制难 | 周期变长，实验室验证压力高 |
| 低损耗材料 | 压合与钻孔窗口不同于标准FR-4 | 良率和来料控制更敏感 |
| 背钻 | 深度和残stub需稳定受控 | 程序、刀具和验证成本提高 |
| 大尺寸板 | 翘曲和机械搬运风险高 | 良率和装配反馈更关键 |
| 高频连接器区 | 局部布线密集、参考层要求严 | 图纸评审和DFM难度上升 |

## SI/PI与制造接口

高速板问题经常出在“设计仿真成立，但制造现实没有跟上”。常见断点包括：

- 阻抗模型基于理想铜厚和介质厚度，而量产波动未纳入。
- 仿真使用名义 `Dk/Df`，而非量产批次窗口。
- 背钻设计合理，但现场残stub验证不足。
- 电源平面设计成立，但铜厚、压降和温升验证不够。

因此，高速板导入应把SI/PI要求转成制造端的 `coupon`、TDR/VNA、切片和SPC计划。

## 典型失效与风险

| 风险 | 常见表现 | 排查重点 |
| --- | --- | --- |
| 阻抗偏移 | 通道裕量不足、误码率上升 | 线宽、介质厚度、铜厚、材料批次 |
| 插损超标 | 高速链路不稳定 | 材料Df、铜粗糙度、表面处理、通道长度 |
| 背钻异常 | 反射、眼图恶化 | 程序版本、钻深、切片和X-ray |
| 电源压降/热点 | 高载荷下不稳定 | 铜厚、平面设计、热仿真与实测差异 |
| 板弯/连接器失配 | 装配应力或插拔问题 | 板厚、加强件、散热器受力、运输条件 |

## 可靠性与验证重点

高速板除了常规PCB验证外，还应重点关注：

- TDR / VNA 通道验证。
- 背钻截面与残stub验证。
- 高温运行下的插损、阻抗和供电稳定性。
- 连接器插拔寿命和机械应力。
- 大型散热件安装后的板弯和翘曲。

## 运营与尽调重点

看高速板企业时，重点不只在“能不能做高速板”，而在以下五点：

1. 是否有稳定量产的低损耗材料经验。
2. 是否能把SI/PI要求转成制造和验证闭环。
3. 是否具备大尺寸高层数板的稳定良率。
4. 是否有背钻、阻抗和高速测试的工程能力。
5. 收入和毛利是否体现出高速板产品结构，而不是仅停留在送样或宣传阶段。

## 高速板项目评审清单

1. 目标速率、损耗预算和阻抗窗口是否明确。
2. `stack-up` 是否与量产材料和压合窗口一致。
3. 背钻、stub、连接器和过孔结构是否闭环。
4. `coupon`、TDR/VNA、切片和SPC方案是否已定义。
5. 铜厚、电流、压降和热路径是否已验证。
6. 大尺寸板的搬运、装配和翘曲控制是否考虑充分。
7. 客户验收是否看功能、眼图、误码率、还是结构参数。
