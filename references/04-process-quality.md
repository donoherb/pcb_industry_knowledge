# 制程、良率与质量

## 刚性多层PCB基本流程

典型流程可概括为：

开料 -> 内层图形 -> 内层蚀刻 -> AOI -> 棕化/黑化 -> 叠板压合 -> 钻孔 -> 除胶渣/沉铜 -> 电镀 -> 外层图形 -> 外层蚀刻 -> 阻焊 -> 字符 -> 表面处理 -> 成型 -> 电测 -> 终检 -> 包装。

这是刚性多层板的简化主线，实际流程会随板种、厂内工艺路线和客户要求变化。常见差异包括：

- 先整板电镀再外层图形，或外层图形后图形电镀，再蚀刻形成最终线路。
- 表面处理、字符、成型、电测和终检的先后顺序可因工艺、设备和客户要求调整。
- HDI、高多层、高速背板、厚铜板、金属基板和汽车板会增加背钻、控深钻、埋阻埋容、填孔、电镀加厚、阻抗coupon、可靠性抽样和特殊清洁/烘烤步骤。
- 量产流程图必须覆盖外协、返工、重工、报废、隔离、检验和追溯路径，不能只画主流程。

关键工序控制要点：

| 工序 | 关键控制 | 常见失效 |
| --- | --- | --- |
| 开料/前处理 | 材料批次、铜厚、板材方向、裁切尺寸、清洁度、烘烤条件 | 材料混用、尺寸偏差、吸湿、表面污染 |
| 内层图形 | 曝光能量、对位、显影、干膜/油墨状态、洁净度 | 开路、短路、线宽偏差、残胶、针孔 |
| 内层蚀刻 | 蚀刻速率、喷压、温度、铜离子浓度、补偿值 | 过蚀、欠蚀、侧蚀、阻抗偏差 |
| 压合 | lay-up顺序、树脂流动、升温速率、压力、真空、冷却 | 层偏、空洞、分层、白斑、板厚异常、翘曲 |
| 钻孔 | 钻咀寿命、转速/进给、叠板厚度、定位、除尘 | 孔偏、毛刺、胶渣、孔壁粗糙、内层拉伤 |
| 去钻污/沉铜 | 除胶渣、粗化、活化、化学铜均匀性、孔内润湿 | 孔壁空洞、背光不良、孔铜附着差 |
| 电镀 | 电流密度、搅拌、阳极、添加剂、板面电流分布、夹具 | 孔铜不足、面铜不均、烧焦、夹点缺陷、填孔凹陷 |
| 外层图形/蚀刻 | 对位、干膜解析度、镀层厚度、蚀刻补偿 | 线宽线距不良、短路、开路、铜残留 |
| 阻焊 | 油墨批次、预烘、曝光、显影、固化、对位 | 偏位、桥连、起泡、附着差、阻焊入孔 |
| 表面处理 | 前处理、药水浓度、温度、时间、负载量、清洗和干燥 | 氧化、镀层薄/厚、镍腐蚀、可焊性差、离子污染 |
| 成型/电测/终检 | 程序版本、治具状态、刀具寿命、检验标准、标签追溯 | 尺寸偏差、毛刺、漏测、误判、混料、标签错误 |

## HDI流程要点

HDI强调微孔和高密度互连。典型难点包括：

- 激光钻孔孔形和残胶控制。
- 微孔填孔电镀的空洞、凹陷和均匀性。
- 多次压合带来的对位误差累积。
- 叠孔可靠性和热循环失效。
- 细线路图形转移和蚀刻控制。

任意层HDI、SLP和细线路产品通常需要更强的LDI、mSAP、填孔电镀、对位补偿、清洁度和过程追溯能力。

HDI常见结构和风险：

| 结构/工艺 | 特点 | 质量关注 |
| --- | --- | --- |
| 一阶/二阶HDI | 通过一次或多次增层形成微盲孔 | 激光孔形、底铜清洁、微孔电镀、层间对位 |
| 任意层HDI | 多层之间可通过微孔高密度互连 | 多次压合涨缩累积、叠孔可靠性、良率爬坡 |
| Staggered via错孔 | 微孔错开排列 | 密度略低但可靠性通常优于叠孔 |
| Stacked via叠孔 | 微孔垂直堆叠 | 对填孔质量、界面结合和热循环更敏感 |
| Via-in-pad | 孔位于焊盘内 | 填孔凹陷、镀帽、焊接空洞、平整度 |

HDI失效分析不要只看最终开短路。微孔底部界面裂纹、填孔空洞、树脂凹陷、铜帽裂纹、孔底清洁不足和热循环后阻值漂移，往往需要切片、X-ray/CT、导通电阻监控和可靠性测试联动判断。

## mSAP/SAP流程要点

mSAP/SAP属于半加成或改良半加成细线路路线，通常用于SLP、先进HDI和IC载板。其核心不是简单“蚀刻更细”，而是通过薄底铜、图形电镀和去种子层形成更精细、更垂直的线路。

典型控制点：

| 环节 | 控制重点 | 典型风险 |
| --- | --- | --- |
| 种子层/底铜 | 厚度、均匀性、附着力、表面粗糙度 | 附着不足、残铜、线底粗糙影响信号 |
| 图形形成 | 干膜解析度、曝光/显影、洁净度 | 线距桥连、缺口、侧壁粗糙 |
| 图形电镀 | 铜厚、线形、添加剂、均匀性 | 线宽超差、蘑菇形、夹点和厚度不均 |
| 去膜/闪蚀 | 去除底铜但保留线路 | 底铜残留导致短路，过蚀导致线宽不足 |
| AOI/AVI | 高分辨率检测和真假点复判 | 微短路、残铜、缺口、异物漏检 |

判断企业是否具备mSAP/SAP量产能力，应看量产线宽线距、良率、缺陷密度、洁净度、客户认证、设备配置和长期订单，而不是单片样品或宣传指标。

## FPC流程要点

FPC关注弯折、薄型化和装配适配。重点工序包括PI/FCCL加工、线路形成、覆盖膜贴合、补强、冲切、表面处理和贴装协同。质量风险常见于弯折裂纹、尺寸稳定、胶溢、开窗偏差和焊盘可靠性。

FPC设计和制造要区分动态弯折与静态弯折：

| 场景 | 关注重点 | 典型控制 |
| --- | --- | --- |
| 动态弯折 | 多次往复弯折寿命 | 压延铜、走线方向、弯折半径、单面铜优先、避免孔和焊盘落在弯折区 |
| 静态弯折 | 装配时一次或少数几次弯折 | 补强位置、覆盖膜开窗、应力释放、弯折治具 |
| 贴装区域 | SMT焊接和连接器可靠性 | 补强、平整度、表面处理、焊盘设计、回流温度 |
| 刚柔结合 | 刚性区与柔性区过渡 | 压合空洞、树脂流动、应力集中、开窗边缘裂纹 |

FPC常见失效包括铜箔疲劳裂纹、覆盖膜起泡/分层、胶溢污染焊盘、补强偏位、冲切毛刺、离子污染和ENIG/OSP可焊性衰减。评价FPC能力时，应看弯折寿命、尺寸稳定性、贴装良率和客户装配端反馈。

## IC载板流程要点

IC载板常采用多次build-up工艺，关键难点包括：

- 细线路形成和线路粗糙度控制。
- 微孔形成、填孔、电镀均匀性。
- ABF/BT材料处理和尺寸稳定。
- 大尺寸载板翘曲控制。
- 洁净度和微缺陷控制。
- 与封装工艺的凸点、焊盘、阻焊和表面处理匹配。

载板良率对微缺陷更敏感。小缺陷在PCB上可能可接受，在封装基板上可能导致芯片封装失效。

IC载板与普通PCB的过程控制差异：

| 维度 | 普通PCB | IC载板 |
| --- | --- | --- |
| 缺陷敏感度 | 取决于线宽、层数和应用等级 | 微短路、残铜、颗粒、微孔缺陷会直接影响封装良率 |
| 生产环境 | 高阶产品需要洁净控制 | 更强调洁净室、颗粒、金属污染和微缺陷管控 |
| 尺寸/翘曲 | 影响SMT和装配 | 影响芯片贴装、倒装焊、底填、封装翘曲和共面性 |
| 线路工艺 | 减成法、mSAP等并存 | ABF载板多依赖SAP/mSAP和高精度曝光/电镀 |
| 表面处理 | 焊接、接触、可焊性 | 还要匹配wire bonding、flip chip bump、焊球和封装材料 |
| 可靠性 | IPC/客户可靠性 | 更贴近封装可靠性和JEDEC/客户封装条件 |

载板过程质量应特别关注：ABF/BT材料吸湿和储存、激光孔残渣、除胶渣、微孔金属化、细线路残铜、阻焊/SR开窗、焊盘共面性、warpage、离子/颗粒污染、strip级追溯和封装客户的来料判定规则。

## 良率瓶颈

| 环节 | 常见问题 | 影响 |
| --- | --- | --- |
| 图形转移 | 短路、开路、线宽偏差 | 电气失效、阻抗偏差 |
| 钻孔 | 孔偏、毛刺、孔壁粗糙 | 互连可靠性下降 |
| 沉铜/电镀 | 孔铜不足、空洞、厚度不均 | 热循环和导通失效 |
| 压合 | 层偏、气泡、分层、翘曲 | 结构和可靠性失效 |
| 阻焊 | 偏位、桥连、附着不足 | 焊接和绝缘问题 |
| 表面处理 | 镍腐蚀、金厚不足、氧化 | 焊接、打线和接触失效 |
| 成型 | 尺寸偏差、毛刺、裂纹 | 装配问题 |
| 检测 | 漏检、误判、追溯弱 | 客诉和报废 |

良率分析要区分三个层级：

| 层级 | 指标 | 用途 |
| --- | --- | --- |
| 工序良率 | 单工序通过率、返工率、报废率 | 找出瓶颈工序和设备/班次/材料差异 |
| 一次合格率FPY | 不经返工一次通过的比例 | 反映真实过程成熟度，优于只看最终出货合格率 |
| 最终良率 | 最终合格出货比例 | 反映交付能力，但可能掩盖返工和质量成本 |

高阶PCB和载板应同时看缺陷密度。对细线路和大尺寸载板，面积越大、线路越密、层数越高，单件中遇到关键缺陷的概率越高，因此良率爬坡不能只靠最终抽检，应依赖过程数据分层、缺陷地图、设备/批次追溯和SPC。

AOI/AVI、ET电测、FQC/终检和报废结果如何统一到panel/pcs/strip/unit位置图，详见 `36-defect-scrap-mapping-aoi-et-fqc.md`。

## 常见缺陷、机理与改善方向

| 缺陷 | 可能机理 | 改善方向 |
| --- | --- | --- |
| 开路 | 线路缺口、干膜破损、过蚀、划伤、电测探针问题 | 图形洁净度、曝光显影窗口、AOI、蚀刻补偿、治具点检 |
| 短路 | 残铜、干膜桥连、蚀刻不足、异物、阻焊桥 | 清洁度、显影/蚀刻控制、AOI/AVI复判、洁净室管理 |
| 孔铜不足/空洞 | 除胶渣不足、活化不良、沉铜覆盖差、电镀电流分布不均 | 药水分析、背光测试、切片、夹具和电流密度优化 |
| 分层/起泡 | 材料吸湿、压合不足、棕化/粗化不良、污染 | 材料烘烤、表面处理、压合曲线、储存和清洁控制 |
| CAF | 玻纤/树脂界面缺陷、离子污染、孔间距小、湿热偏压 | 材料选择、钻孔质量、清洗、设计间距、CAF验证 |
| 阻抗偏差 | 线宽、介质厚度、铜厚、Dk偏差、表面粗糙度 | 阻抗模型、coupon、线宽/压合/SPC、材料批次控制 |
| 翘曲 | 叠构不对称、铜分布不均、CTE不匹配、固化/冷却不均 | 叠构平衡、铜平衡、压合曲线、烘烤和治具支撑 |
| ENIG镍腐蚀 | 前处理、镍磷层、浸金置换反应、药水老化 | 药水窗口、金厚/镍厚、截面分析、可焊性和供应商控制 |
| 离子污染 | 清洗不足、药水残留、手汗/环境污染、包装污染 | 清洗验证、ROSE/离子色谱、洁净操作、包装材料控制 |
| 漏检/误判 | 检测分辨率不足、程序错误、人员标准不一致 | MSA、属性一致性、限度样、程序版本和复判规则 |

## 可靠性与测试

常见测试和验证维度：

- 电测：开短路、绝缘、电阻。
- AOI/AVI/X-ray：图形、孔、层偏和外观缺陷。
- 阻抗测试：高速板关键。
- 热冲击/热循环：孔铜、层间和焊盘可靠性。
- HAST/PCT：湿热可靠性。
- CAF测试：绝缘材料和孔间距可靠性。
- 剥离强度：铜箔或镀层附着。
- 焊锡耐热/回流模拟：装配耐受。
- 离子污染：洁净度和电化学迁移风险。
- 翘曲测试：IC载板和封装相关产品关键。

常见测试项目的工程含义：

| 测试 | 主要验证 | 典型失效/风险 | 适用场景 |
| --- | --- | --- | --- |
| 电测 | 开短路、网络连通、绝缘 | 线路开短、孔导通不良、测试治具漏测 | 全部PCB/载板，出货放行基础 |
| AOI/AVI | 图形、外观、阻焊、字符和表面缺陷 | 微短路、缺口、残铜、阻焊桥、划伤 | 内外层、阻焊后、终检 |
| X-ray/CT | 孔位、层偏、填孔、隐藏结构 | 盲孔偏移、填孔空洞、背钻异常 | HDI、载板、背钻、高多层 |
| 微切片 | 孔铜、内层连接、镀层、微孔结构 | 孔铜薄、空洞、裂纹、树脂凹陷、镀层异常 | 过程确认、PPAP、可靠性失效分析 |
| 阻抗/TDR/VNA | 传输线阻抗、损耗、回损 | 线宽/介质/铜厚偏差、材料Dk/Df偏差 | 高速、射频、服务器、通信 |
| 热应力/焊锡耐热 | 短时高温焊接耐受 | 分层、起泡、孔壁裂纹、焊盘起翘 | SMT/波峰焊/返修耐受 |
| 热循环/热冲击 | 热膨胀循环下互连可靠性 | 孔铜疲劳裂纹、微孔界面裂纹、层间失效 | 汽车、工控、服务器、载板 |
| HAST/PCT | 湿热压力环境下绝缘和界面可靠性 | CAF、离子迁移、分层、绝缘下降 | 高可靠、车规、封装相关 |
| CAF | 带偏压湿热下导电阳极丝风险 | 绝缘失效、孔间/线间短路 | 高密度、高电压、高湿应用 |
| 离子污染/离子色谱 | 残留离子和清洁度 | 电化学迁移、腐蚀、漏电 | 高可靠、细间距、表面处理后 |
| 可焊性/润湿 | 表面处理和储存后的焊接能力 | 润湿不足、氧化、污染、镀层异常 | OSP、ENIG、沉银、沉锡、HASL |
| 剥离强度 | 铜箔/镀层与介质附着 | 铜箔剥离、镀层附着差 | 材料认证、压合、电镀验证 |
| 翘曲/共面性 | 板件或载板平整度 | SMT虚焊、封装贴装不良、BGA共面性风险 | 薄板、大尺寸板、载板 |

可靠性测试设计要点：

- 测试条件、样本量、判定准则、前处理和失效定义必须来自客户规范、IPC/JEDEC/内部标准或DVP&R，不能只写“通过可靠性”。
- 可靠性样品应尽量来自代表性量产工艺和材料。工程样、返工样或非量产设备样品不能直接代表PPAP量产能力。
- 破坏性测试应保留样品编号、批次、位置、切片照片、原始数据和失效分析记录。
- 可靠性失败要同时区分设计问题、材料问题、过程问题、测量/制样问题和样本代表性问题。
- 对HDI/载板，建议把导通电阻在线监控、热循环后切片、X-ray/CT和失效定位结合使用，避免只看最终开路。

## 过程质量控制框架

PCB/载板的过程质量控制应从“终检筛选”转向“过程预防”。建议按以下顺序建立控制逻辑：

`客户要求/特殊特性 -> 过程流程图 -> PFMEA -> Control Plan -> SOP/参数窗口 -> MSA -> SPC/过程能力 -> 异常反应 -> 8D/ECN/经验教训`

| 控制层级 | 目的 | PCB/载板示例 |
| --- | --- | --- |
| 来料控制 | 保证材料、药水和外协输入稳定 | CCL/PP/铜箔/油墨/药水CoA、批次追溯、来料检验 |
| 首件确认 | 防止程序、材料、治具和参数初始错误 | LDI首件、钻孔首件、阻焊首件、表面处理首件 |
| 过程参数控制 | 保持关键工序窗口稳定 | 蚀刻速率、电镀电流密度、压合曲线、药水浓度 |
| 在线/离线检测 | 及时发现工序异常 | AOI、线宽、孔铜、镀层、阻抗、离子污染 |
| SPC与能力 | 判断过程稳定性和长期能力 | 控制图、Cpk/Ppk、趋势和异常反应 |
| 防错和权限 | 避免人为、程序和混料错误 | 条码扫描、程序锁定、治具防呆、参数权限 |
| 追溯与隔离 | 快速界定影响范围 | 材料批次、设备、班次、panel/lot、检验记录 |

关键原则：没有可信测量系统，不应使用数据做能力判断；没有稳定过程，不应只用Cpk数值证明过程能力。MSA、SPC和控制计划应与 `27-msa.md`、`28-spc.md`、`26-apqp.md` 联动使用。

## SPC与过程能力应用

PCB/载板适合纳入SPC的项目通常具备三点：影响客户功能/可靠性，能稳定测量，过程有连续数据。

| 项目 | 常用控制对象 | 注意点 |
| --- | --- | --- |
| 线宽线距 | 蚀刻后线宽、线距、残铜 | 注意测点规则、铜厚和AOI/影像测量一致性 |
| 孔铜/面铜 | 孔铜厚、面铜厚、电镀均匀性 | 切片破坏性测试可结合coupon和过程参数SPC |
| 阻抗 | coupon阻抗、关键网络抽测 | 需要材料、线宽、介质厚度和铜厚同步分析 |
| 镀层厚度 | ENIG/ENEPIG/沉银/沉锡等厚度 | XRF程序、标准片和测点要受控 |
| 板厚/介质厚度 | 压合后厚度、介质厚度 | 叠构、树脂流动和铜分布影响显著 |
| 翘曲 | 常温/热态翘曲、共面性 | 测量支撑方式、温湿度和前处理要一致 |
| 离子污染 | ROSE或离子色谱指标 | 样品处理、溶液、温度、时间影响大 |
| 外观/属性缺陷 | AOI/AVI缺陷率、漏判/误判 | 需要属性MSA、限度样和复判规则 |

过程能力常见误区：

- 过程不稳定时计算Cpk/Ppk，数字没有决策意义。
- 用过宽样本范围或混合多种产品计算能力，掩盖具体产品问题。
- 只看最终合格率，不看FPY、返工和报废。
- Cpk达标但中心漂移、趋势恶化或SPC频繁判异，仍说明过程有风险。
- 用不可信量具的数据证明过程能力。

## 检验、判定与追溯

检验系统应覆盖“检测能力、判定标准、人员一致性、数据追溯”四个方面。

| 项目 | 要点 |
| --- | --- |
| 检验标准 | 图纸、客户规范、IPC等级、限度样和内部标准要一致 |
| 检验方法 | 测点、倍率、光照、距离、角度、样本量和记录格式要明确 |
| 人员能力 | 外观、切片、X-ray、AOI/AVI复判等应做培训和一致性确认 |
| 设备程序 | AOI/AVI、电测、XRF、TDR、CMM等程序版本和权限受控 |
| 数据追溯 | 检验记录应能追溯到lot、panel、材料批次、设备、班次和出货批次 |
| 异常隔离 | 不合格范围、可疑批次、在制品、库存和已出货品要可界定 |

对外观类和属性类判定，限度样、Master Sample和属性MSA很关键。人工目检和自动化检测不能只看检出数量，还要关注漏判率、误判率、复判规则和客户判定一致性。

## 常见标准

常见参考体系包括IPC、JEDEC、UL、ISO/IATF和客户企业标准。具体版本会更新，做合规或认证判断时必须核对当前有效版本。

| 标准/体系 | 常见用途 |
| --- | --- |
| IPC-A-600 | PCB外观验收 |
| IPC-6012 | 刚性PCB性能规范 |
| IPC-6013 | 柔性PCB性能规范 |
| IPC-6018 | 高频/微波PCB相关规范 |
| IPC-2221/2222 | PCB设计通用/刚性板设计 |
| IPC-4101 | 覆铜板材料规范 |
| IPC-4552等 | 表面处理相关规范 |
| IPC-TM-650 | 测试方法 |
| IATF 16949 | 汽车供应链质量管理 |
| ISO 9001/14001 | 质量和环境管理体系 |
| UL认证 | 安规和材料体系相关 |

标准使用边界：

- IPC Class 1/2/3是产品性能和可接受性等级逻辑，不能简单等同于“低端/中端/高端报价”。客户图纸、采购文件和质量协议优先。
- `IPC-A-600` 偏可接受性图示和检验参考，`IPC-6012/6013/6018` 偏资格和性能要求；不能只引用A-600替代性能规范。
- `IPC-TM-650` 是测试方法库，不是产品验收规范；需要由图纸、采购文件或性能规范指定具体方法。
- 汽车、航天、医疗、服务器、封装载板等项目通常有客户补充规范或addendum，应与基础标准一起使用。
- 客户冻结旧版标准时，按合同执行旧版；内部知识库中的较新版本不能自动替代。

### 五大工具与六大工具的口径

- 传统中文语境常说“IATF 16949五大工具”：`APQP`、`PPAP`、`FMEA`、`MSA`、`SPC`。
- 当前AIAG Quality Core Tools页面将 `APQP`、`Control Plan`、`PPAP`、`FMEA`、`SPC`、`MSA` 作为汽车质量核心工具展开；2024年AIAG把APQP第3版和Control Plan第1版拆成独立手册。因此实践中可按“六大工具”理解：在五大工具基础上，将 `Control Plan` 单列。
- `Control Plan` 不是孤立文件。它应承接过程流程图和PFMEA的风险识别结果，并把关键产品/过程特性、控制方法、检测频率和反应计划落到量产现场。
- IATF 16949体系审核不能只看证书。内部审核、过程审核和供应商审核人员需要理解与审核范围相关的核心工具要求；具体适用深度还受客户特定要求（CSR）影响。

### 六大工具关系链

典型逻辑链可概括为：

客户要求/图纸/CSR -> APQP项目策划 -> 过程流程图 -> DFMEA/PFMEA -> Control Plan -> MSA确认测量系统 -> SPC监控稳定性和能力 -> PPAP提交量产批准证据 -> 量产问题反馈并更新FMEA/Control Plan/SPC。

在PCB和IC载板项目中，不要把这些工具当作“交付表单”。更重要的是用同一组特殊特性和风险项贯穿设计评审、工艺开发、样品验证、试生产、量产控制、客诉闭环和工程变更。

### 汽车PCB/载板审核关注点

- APQP是否覆盖从报价/DFM到样品、试产、量产、变更的完整项目节点，而不是只在PPAP前补文档。
- 过程流程图、PFMEA、Control Plan、作业指导书和现场记录是否一一对应。若PFMEA识别孔铜不足为高风险，Control Plan中应能看到对应的电镀参数、微切片/孔铜厚测量、抽样频率和异常反应。
- 特殊特性是否在图纸、工艺路线、PFMEA、Control Plan、检验规范、SPC和PPAP证据中贯穿一致。
- MSA是否覆盖关键测量系统。AOI/AVI等属性检验不要只看设备精度，还要看误判/漏判、一致性和复判规则。
- SPC是否用于真正影响客户功能和可靠性的CTQ/CTP，不要只对容易采数但低风险的项目画控制图。
- PPAP是否由实际量产节拍、量产设备、量产人员和量产材料支撑；样品线或工程批数据不能直接代表量产能力。
- 客户CSR优先。GM、Ford等OEM会更新APQP、Control Plan、PPAP、特殊过程评估和系统上传要求；Tier 2/Tier 3 PCB供应商也可能被逐级传递。

## IPC检验与验收规范速查

核验日期：2026-04-17。来源优先级：IPC/Global Electronics Association官方商店、官方标准状态页和官方新闻稿。此处只沉淀标准定位、版本和选用逻辑，不复制IPC付费标准的条款、图片或验收表格；正式判定必须以客户图纸/采购文件、质量协议和已授权的标准原文为准。

### 选用逻辑

- 裸板制造和来料检验：先按产品类型选 `IPC-601X` 性能规范，再用 `IPC-A-600` 做可视化外观/内部缺陷验收参考；电测、焊接性、表面处理、储存和测试方法需分别引用专项标准。
- 刚性PCB：通常以 `IPC-6012` 规定资格、性能、测试频率和质量一致性要求，`IPC-A-600` 作为外观和剖面缺陷图示化验收参考。
- FPC/软硬结合：通常以 `IPC-6013` 规定柔性和刚柔结合板性能要求，注意覆盖膜、补强、刚柔过渡区、铜箔/孔铜和弯折相关风险。
- 高频/微波PCB：通常以 `IPC-6018` 覆盖高频微波板的材料、表面处理、导体、孔/微孔、测试频率和电气/机械/环境可靠性要求。
- 有机IC载板：`IPC-6921` 是2025年新增的有机IC载板要求与验收标准，适用于wire bonding和flip chip有机IC载板；比一般PCB更强调功能区差异化判定、细线路/微孔/表面处理、质量一致性和可靠性。
- PCBA检验：`IPC-A-610` 是电子组件外观验收标准，应与焊接工艺和材料要求 `IPC J-STD-001` 配合使用；线束另用 `IPC/WHMA-A-620`。
- 汽车、航天、军工、通信等场景：不要只引用基础标准，应确认采购文件是否要求相应addendum。addendum通常不能单独使用，必须与基础标准同用。
- 客户指定旧版时，按合同/图纸执行旧版；内部知识库中的“最新版本”不能自动替代客户已冻结的版本。

## 质量分析方法

判断企业制造质量时，不要只看“通过认证”。还要看：

- 认证对应的产品、客户和工厂是否一致。
- 量产良率、客户退货率、报废率和客诉趋势。
- 是否具备过程追溯和统计过程控制。
- 新产线爬坡的良率曲线和问题闭环。
- 质量成本是否体现在毛利率、存货跌价、售后费用或客户赔偿中。

更完整的质量尽调或客户审核可按以下维度展开：

| 维度 | 关键问题 | 证据 |
| --- | --- | --- |
| 产品匹配 | 被审核产线是否生产目标产品，而不是只展示成熟低难度产品 | 产品清单、客户清单、产线布局、出货记录 |
| 工艺能力 | 线宽线距、孔径、纵横比、层数、阻抗、表面处理是否稳定量产 | 能力矩阵、良率、Cpk/Ppk、客户认证 |
| 质量体系 | APQP、PFMEA、Control Plan、MSA、SPC、8D、ECN是否真正运行 | 项目资料、控制计划、SPC图、8D和变更记录 |
| 过程追溯 | 能否从出货批追到材料、设备、人员、参数、检验和返工 | MES/QMS、批次记录、条码、设备数据 |
| 实验室能力 | 切片、XRF、TDR、离子污染、可靠性测试是否受控 | 设备清单、校准、MSA、实验室资质、原始数据 |
| 客户质量 | 客诉、退货、PPM、重复问题和8D关闭质量 | 客户评分卡、8D、退货分析、纠正验证 |
| 供应商质量 | 关键材料和外协过程是否纳入质量管理 | 供应商清单、来料趋势、供应商8D、变更通知 |
| 变更管理 | 材料、设备、参数、地点、外协和软件程序变化是否受控 | ECN、客户批准、再验证、PPAP影响评估 |

PCB/载板质量问题的分析路径：

1. 明确问题定义：产品、批次、数量、客户使用场景、失效现象、规格和判定标准。
2. 隔离影响范围：库存、在制、在途、客户现场和同材料/同设备/同班次批次。
3. 分层数据：按产品、层数、材料、设备、班次、工序、供应商、时间和缺陷位置拆分。
4. 复核测量系统：确认检验方法、量具、人员、限度样和客户判定是否一致。
5. 找发生原因和流出原因：发生原因解释缺陷怎么产生，流出原因解释为什么没有拦住。
6. 验证纠正措施：通过小批、SPC、能力、可靠性或客户确认验证有效。
7. 标准化：更新PFMEA、Control Plan、SOP、检验规范、培训、供应商要求和经验教训库。

## 良率爬坡阶段管理

良率不是单一数字，而是不同阶段的不同管理对象。工程样、试产和成熟量产看同一组指标，容易得出错误结论。

| 阶段 | 关注重点 | 关键指标 | 典型管理动作 |
| --- | --- | --- | --- |
| 工程样/样件 | 缺陷机理识别和制程可行性 | 关键缺陷类型、样品通过率、设计可制造性问题 | 快速切片、FA、设计和工艺同步修正 |
| 试生产 | 流程完整性、追溯、MSA和初始控制计划 | FPY、返工率、关键特性分布、异常闭环速度 | 建立批次追溯、验证反应计划、补齐控制计划 |
| 量产爬坡 | 班次差异、设备负荷、药水负载和瓶颈工序稳定性 | 小时产出、缺陷密度、停机率、Cpk/Ppk趋势 | 按班次/设备/材料分层，做Safe Launch或加严控制 |
| 成熟量产 | 波动收敛、成本和逃逸风险控制 | 客诉PPM、COPQ、SPC判异率、长期可靠性 | 优化参数窗口、降低报废返工、固化经验教训 |

对高阶HDI、mSAP和IC载板，建议把“关键缺陷密度”和“边界样品判定一致性”与传统良率指标一起看，否则最终良率可能掩盖潜在逃逸。

## 失效分析闭环

失效分析的目的不是找到一张漂亮切片，而是把问题从个案证据转成过程控制。

建议闭环顺序：

1. 固定失效定义：确认客户现象、判定标准、复现条件和失效位置。
2. 保留证据链：锁定样品编号、批次、panel位置、材料批次、设备和程序版本。
3. 联合分析：把电测、AOI/AVI、X-ray/CT、微切片、化学分析、可靠性和测量系统一起看，避免只凭单一证据下结论。
4. 区分发生原因和流出原因：前者解释缺陷为何产生，后者解释为何没有被检测、隔离或阻断。
5. 回写控制系统：更新PFMEA、Control Plan、SOP、限度样、培训、供应商要求和相似产品检查表。

高风险缺陷如孔铜不足、CAF、ENIG异常、细线路残铜、载板warpage和客户装配型失效，通常都需要“失效机理 + 过程参数 + 测量系统 + 客户场景”四条证据同时成立，结论才稳。
