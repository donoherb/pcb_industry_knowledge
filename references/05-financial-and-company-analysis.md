# 财务与公司分析

## 公司分析框架

研究PCB或IC载板公司时，按以下顺序拆解：

1. 产品结构：收入来自哪些板种、层数、应用和客户。
2. 产业链位置：是材料、设备、PCB制造、载板制造、EMS配套还是封装供应链。
3. 技术能力：量产指标、认证客户、良率、关键设备和材料。
4. 客户结构：终端行业、客户集中度、认证壁垒、订单能见度。
5. 产能结构：工厂、产品段、稼动率、扩产节奏、爬坡风险。
6. 财务质量：毛利率、费用率、现金流、周转、资本开支、折旧。
7. 风险：周期、价格、客户、材料、环保、汇率、技术替代。

## 关键财务指标

| 指标 | 解释 | 关注点 |
| --- | --- | --- |
| 收入增速 | 量、价、结构和并表影响 | 拆分周期恢复还是份额提升 |
| 毛利率 | 产品结构、稼动率、价格、材料、良率 | 是否受新产线折旧拖累 |
| 净利率 | 毛利、费用、财务、税率综合结果 | 是否有一次性损益 |
| 经营现金流 | 利润转化为现金能力 | 应收、存货和预付款变化 |
| 存货周转 | 需求和生产效率 | 跌价风险、备货策略 |
| 应收周转 | 客户账期和回款 | 大客户议价和坏账风险 |
| 资本开支 | 扩产和设备投入 | 是否匹配订单和认证节奏 |
| 折旧摊销 | 产线投产后的固定成本 | 新产能爬坡期利润压力 |
| ROIC/ROE | 资本回报 | 高资本开支行业必须关注 |
| 研发费用率 | 工艺和产品投入 | 要结合成果和客户导入 |

## 经营指标

优先收集以下非财务指标：

- 各产品收入占比和毛利率。
- 主要客户和终端应用占比。
- 产能、产量、销量、稼动率。
- ASP或单位面积收入。
- 良率、报废率、客诉率。
- 新客户认证进度和量产时间。
- 新产线建设、设备到厂、试产、量产节点。
- 原材料成本占比和价格联动机制。

如果公开资料没有披露，应明确“未披露”，不要用行业平均替代公司事实。

## PCB公司财务特征

成熟PCB企业的利润弹性常来自产品结构升级、稼动率修复、材料成本下降和费用控制。低端板厂容易受到价格竞争和客户转单影响。高端板厂虽然ASP和毛利率更高，但需要持续资本开支、工程能力和客户认证。

判断PCB公司质量时，要看毛利率穿越周期的稳定性，而不是只看景气高点。

## IC载板公司财务特征

IC载板企业资本开支强度高，扩产周期和客户认证周期长。新产线早期可能出现收入增长但利润不显著改善，原因包括折旧上升、试产费用、良率爬坡和客户认证尚未完成。

对载板企业，重点看：

- ABF/BT产品结构。
- 客户和封装平台认证。
- 大尺寸、细线路、低翘曲量产能力。
- 良率爬坡和设备瓶颈。
- 与先进封装、AI/HPC、存储或通信需求的绑定程度。

## 红旗信号

- 扩产口径频繁变化，但客户认证或订单证据不足。
- 宣称高端产品突破，但收入结构和毛利率没有体现。
- 存货持续上升且收入未同步增长。
- 应收账款增长显著快于收入。
- 大客户集中度高且议价能力弱。
- 产能投放后稼动率低，折旧压制长期化。
- 研发资本化或一次性收益显著抬高利润。
- 环保、安全、质量客诉或重大诉讼频发。

## 公司研究模板

### 1. 结论先行

- 公司处于产业链的哪一段。
- 核心优势和主要短板。
- 未来一到三年的关键变量。
- 需要继续核验的事实。

### 2. 业务和产品

- 收入按产品、应用、区域和客户拆分。
- 产品技术层级和竞争对手。
- 量产能力与样品能力的区别。

### 3. 客户和需求

- 主要客户、终端行业和订单能见度。
- 客户认证周期、切换成本和份额变化。
- 需求来自行业增长、份额提升还是库存周期。

### 4. 产能和扩产

- 各工厂产能、产品定位和投产时间。
- 设备到厂、试产、认证、满产节点。
- 资本开支、折旧和融资安排。
- 爬坡风险和产能消化路径。

### 5. 财务拆解

- 收入 = 量 x 价 x 产品结构 x 汇率/并表影响。
- 毛利率 = 产品结构 + 稼动率 + 材料成本 + 良率 + 折旧。
- 现金流 = 利润 + 折旧摊销 - 营运资金增加 - 资本开支。

### 6. 估值和风险

- 与同类公司比较时，口径要一致：产品、客户、成长阶段、资本开支和会计期间。
- 风险至少覆盖需求、价格、良率、客户、材料、环保、汇率和技术替代。

## 访谈提纲

- 当前主力产品的线宽线距、层数、孔径和良率处于什么水平。
- 高端产品收入占比和客户认证情况。
- 新项目从送样到量产通常需要多久。
- 新产线折旧和良率爬坡对毛利率影响多大。
- 上游材料是否有单一供应或客户指定。
- 订单能见度、库存水平和客户拉货节奏如何。
- 公司认为最稀缺的能力是设备、材料、工艺人才还是客户认证。

## 收入和毛利桥接

PCB/载板公司的财务变化最好拆成“量、价、结构、成本、稼动率、折旧”六根桥，而不是只看收入增速。

| 驱动项 | 重点看什么 | 常见误判 |
| --- | --- | --- |
| 出货量 | 面积、片数、panel数、封装基板数量是否一致 | 把订单恢复误判为长期份额提升 |
| ASP | 产品升级、材料传导、客户结构和汇率 | 把铜价或材料转嫁当成真实提价能力 |
| 产品结构 | 高多层、HDI、高速板、ABF/BT占比变化 | 只看高端产品突破新闻，不看收入占比 |
| 良率与报废 | 新工艺、新客户、新产线对制造成本的影响 | 只看最终良率，不看返工和隐性质量成本 |
| 稼动率 | 固定成本摊薄、班次效率、瓶颈工序负荷 | 用整体稼动率掩盖高端产线利用不足 |
| 折旧与资本开支 | 新厂投产节奏、设备验收、满产路径 | 认为扩产一定带来利润同步增长 |

分析毛利率时，最有价值的问题通常是：改善来自可持续的产品和客户升级，还是来自周期修复与成本顺风。

## 同业比较口径

同业对比失真，大多是因为口径没有对齐。最低限度应统一：

- 产品段：标准板、HDI、高多层、高速板、FPC、BT、ABF不能混成一类。
- 应用端：消费、汽车、通信、服务器、封装链的盈利结构不同。
- 阶段：成熟产线和新爬坡产线不应直接比较净利率。
- 公司边界：集团、上市主体、子公司和合资公司的口径要区分。
- 会计口径：财年/自然年、币种、并表范围、一次性损益和政府补助要拆开。

若这些口径无法统一，宁可做“定性对标 + 局部指标比较”，也不要给出看似精确但实际不可比的排序。

## 经营韧性判断

对周期行业，公司好坏不仅取决于景气上行时赚多少，还取决于下行期能否扛住。可重点看：

- 现金流是否能覆盖扩产和营运资金波动。
- 客户和产品是否分散，能否避免单一终端波动冲击。
- 高端产品收入是否经过多个周期验证，而不是短期导入。
- 新产线、老产线和低端产线之间是否存在内部资源挤压。
- 管理层是否在扩产、降价、客户导入和资本回收之间保持一致口径。
