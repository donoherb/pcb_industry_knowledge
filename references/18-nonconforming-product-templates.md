本文提供不合格品管理的现场模板骨架，适用于PCB、HDI、FPC、软硬结合板、载板及相关来料、在制、成品、外协和客户退货场景。正式使用时仍应结合企业QMS字段、MES/QMS系统状态和客户批准流程。

## 使用原则

- 主方法论看 `17-nonconforming-product-management.md`。
- 本页解决“记录怎么写、审批怎么留痕、放行怎么闭环”。
- 模板必须与实际实物流、系统流和审批权限一致，不能只停留在文档层。

## 模板总览

| 模板 | 用途 | 适用节点 |
| --- | --- | --- |
| 不合格品报告单 | 初始识别和隔离 | 发现异常时 |
| 隔离与范围界定单 | 界定影响范围 | 初判后 |
| MRB评审单 | 做处置判断 | MRB阶段 |
| 返工申请与放行单 | 管理返工和再检 | 返工路径 |
| 让步/偏差申请单 | 管理让步和偏差 | 让步/偏差路径 |
| 报废审批单 | 管理报废和系统扣账 | 报废路径 |
| 恢复放行确认单 | 记录恢复放行条件 | 返工/筛选后 |
| 客户通知摘要单 | 记录客户通知和批准 | 涉及客户时 |

### 使用要点

- 缺陷描述写现象，不要先写推定根因。
- 批次、数量和状态必须能对应现场实物。

## PCB/载板字段补充建议

模板中建议优先保留以下字段：

- 产品/料号/版本/叠构
- lot/panel/strip/batch
- 机台/线别/班次
- 材料批次/供应商
- 程序版本/AOI程序/TDR程序/治具编号
- 缺陷类型
- 隔离范围
- MRB结论
- 客户批准状态

## 使用误区

- 只有不合格品单，没有隔离范围记录。
- MRB结论只写“特采”或“返工”，不写条件。
- 返工后没有单独再检和放行记录。
- 让步申请没有数量边界和有效期。
- 报废审批与系统扣账脱节。

## 推荐使用顺序

`不合格品报告单 -> 隔离与范围界定单 -> MRB评审单 -> 返工/让步/报废路径模板 -> 恢复放行确认单 -> 客户通知摘要单`
