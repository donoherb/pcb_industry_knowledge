## 定位

本文件聚焦FPC和软硬结合板。它们与普通刚性板最大的区别，不只是基材柔性，而是设计、制程、装配和使用应力高度耦合，很多问题在客户端装配和寿命阶段才暴露。

## 产品边界

| 产品 | 特征 | 典型应用 |
| --- | --- | --- |
| 单面/双面FPC | 轻薄、可弯折 | 手机、显示、摄像头、可穿戴 |
| 多层FPC | 更高布线密度 | 模组互连、车载、工业 |
| 软硬结合板 | 刚性区+柔性区一体化 | 高可靠小型化模组、医疗、航空、汽车 |
| 动态弯折FPC | 长期往复弯折 | 折叠机构、打印头、车载活动结构 |
| 静态弯折FPC | 装配时少数几次弯折 | 消费电子和模组连接 |

## 为什么FPC/软硬结合板难

难点不在“线路更细”本身，而在于以下耦合：

- 材料柔性与尺寸稳定性的矛盾。
- 弯折寿命与布线/开窗/补强设计耦合。
- 覆盖膜、胶系、补强和贴装工艺彼此牵动。
- 装配端回流、治具和连接器应力会放大结构缺陷。

## 核心设计规则

| 主题 | 常见原则 | 失效风险 |
| --- | --- | --- |
| 弯折区布线 | 尽量沿弯折方向、避免直角和应力集中 | 铜疲劳裂纹 |
| 弯折区禁布 | 尽量避免孔、焊盘、开窗边缘和厚补强 | 裂纹、剥离、断线 |
| 补强设计 | 服务贴装和连接器区域，但避免侵入弯折敏感区 | 应力集中、贴装不稳 |
| 覆盖膜开窗 | 控制开窗尺寸和位置，避免边缘应力集中 | 起泡、裂纹、胶溢 |
| 软硬过渡区 | 过渡长度、压合结构和开窗设计要受控 | 分层、裂纹、树脂空洞 |

## 关键材料

FPC/软硬结合板常见关键材料包括：

- `PI` / `FCCL`
- 压延铜 `RA copper`
- 电解铜 `ED copper`
- 覆盖膜和胶系材料
- 补强材料
- 表面处理材料

材料选择不能只按单项性能判断，而要看其与弯折寿命、贴装、压合和尺寸稳定性的综合平衡。

## 制程难点

| 工序 | 关注点 | 常见问题 |
| --- | --- | --- |
| 材料裁切与对位 | 薄材料易变形 | 尺寸偏差、对位不稳 |
| 线路形成 | 薄铜和柔性基材处理 | 线宽偏差、划伤、残铜 |
| 覆盖膜贴合 | 贴合位置、胶流和气泡控制 | 起泡、胶溢、开窗偏位 |
| 补强贴合 | 位置和平整度 | 贴装不良、应力集中 |
| 软硬压合 | 树脂流动和界面结合 | 分层、空洞、裂纹 |
| 成型/冲切 | 毛刺和边缘质量 | 裂口、尺寸偏差、外观不良 |

## 动态弯折与静态弯折差异

| 场景 | 关注重点 | 判断逻辑 |
| --- | --- | --- |
| 动态弯折 | 寿命、铜型、弯折半径、循环次数 | 以失效寿命和裂纹机理为核心 |
| 静态弯折 | 装配可操作性和一次弯折安全性 | 以装配适配和初期可靠性为核心 |

很多项目的问题在于把静态弯折规则套到动态弯折场景，导致寿命严重高估。

## 装配协同

FPC/软硬结合板的问题，很多不是出厂就能完全暴露，而是在客户端贴装、折弯、锁附和整机组装中暴露。

重点协同项包括：

- 回流温度曲线与补强/覆盖膜兼容性。
- SMT吸嘴和治具对薄板平整度的要求。
- 连接器插拔和锁附应力。
- 模组装配后的弯折路径和应力释放空间。

因此，FPC企业能力评估必须看客户端装配反馈，而不只看出厂良率。

## 常见失效机理

| 失效 | 典型诱因 | 场景 |
| --- | --- | --- |
| 铜箔疲劳裂纹 | 动态弯折、铜型不当、弯折半径过小 | 动态FPC |
| 覆盖膜起泡/分层 | 贴合不良、回流热应力、吸湿 | FPC/软硬板 |
| 胶溢污染焊盘 | 开窗和贴合控制不足 | SMT区 |
| 补强偏位 | 对位和治具不稳定 | 连接器/贴装区 |
| 软硬过渡裂纹 | 压合结构和应力集中 | 软硬结合板 |
| 尺寸漂移 | 材料稳定性和工艺张力问题 | 精密模组装配 |

## 验证重点

FPC/软硬结合板专项验证通常应强调：

- 动态或静态弯折验证。
- 回流/装配后结构和外观验证。
- 连接器插拔、锁附和保持力相关验证。
- 软硬过渡区切片和界面分析。
- 客户端装配后再验证，而不是只测裸板。

## 运营与经营特征

| 特征 | 含义 |
| --- | --- |
| 多品种小批量常见 | 换线、首件和工程释放效率重要 |
| 客户协同深 | 设计和装配问题经常要共同解决 |
| 表面看良率不低 | 但客户端装配损失可能很高 |
| 人工和工程经验占比高 | 现场纪律和工艺know-how重要 |

## 常见误判

- 把FPC看成“更薄的PCB”，忽略弯折和装配应力。
- 只看出厂外观和电测，不看客户端装配反馈。
- 把样品通过当成动态弯折寿命已验证。
- 用普通刚性板的成本和节拍逻辑判断软硬结合板。
- 忽略覆盖膜、补强和连接器区域对整体失效的主导作用。

## FPC/软硬结合板专项尽调清单

1. 主力产品是静态弯折还是动态弯折。
2. 是否大量依赖客户端装配补救才能通过。
3. 覆盖膜、补强和软硬过渡区是否有稳定过程窗口。
4. 客户端最常见的投诉是外观、电测、贴装还是寿命。
5. 弯折寿命验证是否使用真实装配状态和真实路径。
6. 良率统计是否包含客户端装配损失和退货反馈。
7. 高价值项目的盈利是否被复杂装配协同和质量成本吞噬。
