## 定位与来源

本页沉淀MSA（Measurement Systems Analysis，测量系统分析）的实务框架，重点用于PCB、IC载板、PCBA和汽车电子项目中的量具、检验方法、属性判定和测试系统确认。


## 一句话理解

MSA回答的问题不是“量具有没有校准”，而是“这个测量系统产生的数据能不能支持产品放行、SPC判异、过程能力评价、PPAP提交和客户争议判定”。

测量系统不是单一量具，而是下列要素的组合：

| 要素 | 含义 | PCB/载板示例 |
| --- | --- | --- |
| 仪器/设备 | 实际产生读数或判定的设备 | XRF、TDR、二次元、AOI/AVI、显微镜、CMM、离子污染测试仪、阻抗测试仪 |
| 夹具/治具 | 定位、支撑、夹持、探测接触的装置 | 阻抗探针治具、翘曲支撑治具、微切片夹具、板厚测量支架 |
| 标准/基准 | 用于校准、比对或赋值的参考 | 标准片、标准电阻/阻抗件、厚度标准、客户限度样、Master Sample |
| 方法/SOP | 测量点、测量顺序、环境、记录和判定规则 | 线宽取点规则、镀层测点、微切片制样方法、外观照明/距离/角度 |
| 软件/算法 | 数据处理、图像识别、滤波和判定逻辑 | AOI算法、XRF程序、TDR算法、图像边缘识别、自动判级模型 |
| 人员 | 操作者、检验员、评价人和判定人 | 外观检验员、实验室技师、阻抗测试员、微切片分析员 |
| 环境 | 温湿度、光照、振动、清洁度、电磁干扰等 | 翘曲/尺寸测量温湿度、显微镜照明、实验室清洁度、设备预热 |
| 被测零件 | 样本本身的形状、表面、位置和过程分布 | 板弯板翘、板边毛刺、铜面粗糙、阻焊反光、批次差异 |

核心逻辑：

`测量值 = 参考值/真值 + 测量误差`

测量误差要从两个方向拆解：

- 位置误差：测得准不准，主要看偏倚、线性、稳定性。
- 宽度误差：测得稳不稳、一致不一致，主要看重复性、再现性和GRR。

## 与IATF、APQP、PPAP、SPC的接口

MSA是六大质量工具中承接现场数据可信度的工具。没有可信测量数据，SPC控制图、Cpk/Ppk、PPAP证据和客诉分析都会失去基础。

| 接口 | MSA作用 |
| --- | --- |
| IATF 16949 | 对控制计划中识别的检验、测量和试验设备系统做统计研究，优先覆盖关键产品/过程特性 |
| APQP | 第四阶段产品和过程验证的重要输出，用于证明量产测量系统可用 |
| PPAP | 新量具、改进量具、测量和试验设备需要提供变差研究证据 |
| SPC | 控制图和过程能力分析之前要确认测量系统有足够分辨力、稳定性和变差水平 |
| Control Plan | 每个评价/测量技术都应明确测量系统、频率、样本、反应计划和MSA状态 |
| 8D/客诉 | 客户与内部测量结果不一致时，MSA是判定争议和根因分析的基础 |

在APQP链路中，MSA通常位于：

`特殊特性 -> 过程流程图/PFMEA -> Control Plan -> MSA -> SPC/过程能力 -> PPAP -> 量产监控`

## 数据类型与方法选择

MSA先按数据类型选方法。

| 数据类型 | 含义 | PCB/载板示例 | 常用MSA方法 |
| --- | --- | --- | --- |
| 计量型数据 | 可得到连续数值，并能与规格或参考值比较 | 线宽线距、孔铜厚度、镀层厚度、阻抗、板厚、翘曲、介质厚度、离子污染、剥离强度 | 分辨率检查、偏倚、线性、稳定性、GRR、ndc |
| 计数/属性数据 | 只能给出类别或合格/不合格判定 | 外观缺陷判定、AOI真假点、AVI判级、通止规、微切片缺陷分类、可焊性等级 | 属性一致性、Kappa、有效性、漏判率、误判率、信号探测法 |
| 破坏性测量 | 测后样品不可重复测同一状态 | 微切片、剥离强度、热应力后截面、部分可靠性验证 | 嵌套型GRR、替代样本设计、同质样本分组、方法验证 |

不要用“检定/校准合格”替代MSA。校准回答设备是否可溯源和是否偏离标准，MSA回答整个测量过程在真实使用条件下是否足够可信。

## MSA与相近概念边界

MSA经常和校准、方法验证、测量不确定度、实验室比对混在一起。实际审核和项目提交时，应先把边界说清楚。

| 概念 | 回答的问题 | 与MSA的关系 | PCB/载板注意点 |
| --- | --- | --- | --- |
| 校准/检定 | 设备相对标准是否偏离、是否可溯源 | 是MSA前提之一，但不能替代MSA | XRF、TDR、二次元、CMM要确认校准范围覆盖实际测量范围 |
| 点检/期间核查 | 设备日常状态是否仍在受控范围 | 支撑稳定性和放行前确认 | 阻抗标准件、厚度标准片、AOI金板、光源照度可纳入点检 |
| 方法验证 | 方法是否适合该材料、结构和缺陷类型 | 是MSA的重要输入，尤其用于破坏性/化学/可靠性测试 | 微切片、离子污染、剥离强度、CAF/HAST不能只看设备能力 |
| 测量不确定度 | 测量结果的可信区间有多宽 | 更偏实验室和合规表达，MSA更偏过程数据可用性 | 客户争议、第三方报告和边界判定时要考虑不确定度 |
| 实验室比对 | 不同实验室或不同系统结果是否一致 | 可作为偏倚、再现性或相关性确认 | 内部工厂、客户实验室、外部实验室的切片/XRF/TDR结果要定期比对 |
| 过程能力研究 | 过程能否稳定满足规格 | 必须建立在测量系统可信的基础上 | Cpk/Ppk异常时先排除测量系统问题，再解释制程波动 |

## 关键统计特性

| 特性 | 核心问题 | 典型判断 |
| --- | --- | --- |
| 分辨率 | 最小刻度能否看出过程变化 | 通常要求分辨率不大于公差或过程变差的1/10；过程控制优先看过程变差 |
| 灵敏度 | 输入变化能否产生有用输出 | 与仪器设计、保养、预热、环境、操作者技能相关 |
| 偏倚 Bias | 测量平均值与参考值差多少 | 偏倚是系统误差；偏倚过大时数据整体偏高或偏低 |
| 线性 Linearity | 全量程内偏倚是否一致 | 小尺寸准、大尺寸不准，或不同厚度/阻抗区间偏差不同，说明线性有风险 |
| 稳定性 Stability | 随时间是否漂移 | 用控制图跟踪同一标准件或稳定样件；关注趋势、超限、周期性漂移 |
| 重复性 Repeatability | 同一人、同一设备、同一零件多次测是否一致 | 常称EV，但不只来自设备，也包括固定条件下的内部变差 |
| 再现性 Reproducibility | 不同人、不同条件测同一特性是否一致 | 常称AV；手动测量和外观判定尤其容易受人员影响 |
| GRR | 重复性与再现性的联合变差 | 判断测量系统变差相对总变差或公差是否可接受 |
| ndc | 测量系统可区分的过程类别数 | 常用要求是 `ndc >= 5`，否则不适合过程控制 |
| Kappa | 属性判定一致性 | 评价人与评价人、评价人与标准之间的一致程度 |

## MSA实施时机

触发MSA的常见时机：

- 新产品、新过程、新特殊特性导入。
- 新量具、新检验设备、新测试程序、新夹具或新软件算法上线。
- 量具维修、校准后发现异常、搬迁或重大保养后。
- 新检验员、评价人或关键岗位换人。
- 客户、供应商、内部实验室之间测量结果不一致。
- 控制计划、测量方法、抽样频率、规格或客户CSR变化。
- 易磨损、易漂移或对环境敏感的量具按周期复核。
- PPAP、客户审核、内审、过程审核或年度验证要求。

## MSA计划

MSA计划应从控制计划和客户要求反推，不应临近PPAP再补表。

计划至少明确：

| 项目 | 要点 |
| --- | --- |
| 测量对象 | 产品特性、过程特性、特殊特性和对应规格 |
| 测量系统 | 设备编号、夹具、软件版本、标准件、方法编号 |
| 数据类型 | 计量型、属性型、破坏性/非破坏性 |
| 研究项目 | 分辨率、偏倚、线性、稳定性、GRR、Kappa等 |
| 样本设计 | 零件数量、覆盖范围、是否代表真实过程变差 |
| 评价人 | 真实使用该系统的检验员/操作员，通常2-3人或按客户要求 |
| 重复次数 | 计量型常见2-3次；属性型常见每人多次重复判定 |
| 盲测与随机 | 评价人不应知道样本标准答案或前次结果，顺序应随机 |
| 接收准则 | AIAG/MSA手册、客户CSR、内部标准和项目风险 |
| 责任与节点 | 责任部门、完成日期、评审人、失效反应计划 |

文章图片中的训练材料给出的常见研究组合可理解为学习口径：

| 方法 | 常见样本/人员配置 | 关注点 |
| --- | --- | --- |
| 稳定性 | 1件稳定样件或标准件，长期多次测量 | 控制图是否受控，是否漂移 |
| 偏倚 | 1件参考值明确的样件，1人多次测量 | 偏倚是否显著，置信区间是否包含0 |
| 线性 | 多个覆盖量程的参考样件 | 偏倚是否随量程变化 |
| GRR | 多件零件、2-3名评价人、2-3次重复 | EV、AV、GRR、ndc |
| Kappa/属性一致性 | 多名评价人、多件样品、多次判定 | 评价人间一致性、与标准一致性、误判/漏判 |
| 信号探测法 | 多名评价人、多件样品、多次判定 | 灰区宽度、属性判定风险与公差/过程变差比较 |

## 研究设计原则

MSA研究设计的核心是代表真实使用场景，而不是把数据做得好看。样本、评价人、设备、环境和方法都应尽量贴近量产控制计划。

| 设计项 | 推荐做法 | 错误做法 |
| --- | --- | --- |
| 样本范围 | 覆盖实际过程变差、规格边界和典型缺陷形态 | 只选标准件、稳定样件或外观最好的样件 |
| 样本数量 | 计量型常见10件左右；属性型应覆盖合格、不合格和边界样 | 样本太少，无法暴露人员差异和灰区 |
| 评价人 | 选择实际执行该测量/判定的人员，覆盖班次或工厂差异 | 只让实验室高手或工程师测量 |
| 重复次数 | 计量型常见2-3次；属性型可多轮重复判定 | 单次测量后直接计算结论 |
| 随机与盲测 | 样本顺序随机，隐藏标准答案和前次结果 | 按编号顺序测量，评价人知道样本状态 |
| 环境条件 | 使用量产或客户要求的温湿度、光照、预热、清洁和夹持条件 | 在临时优化条件下完成研究，量产现场无法复现 |
| 数据记录 | 保留原始读数、判定、异常备注、软件版本和测量顺序 | 只保存统计软件截图或最终结论 |

样本覆盖范围要和MSA目的匹配：

- 用于产品放行时，样本应覆盖规格上下限、临界合格和临界不合格区域。
- 用于SPC或过程能力时，样本应代表稳定量产过程的实际变差。
- 用于客户争议或实验室比对时，应优先选择双方可重复测量、参考值明确或可达成共识的样本。
- 用于新设备导入时，应覆盖设备全量程、不同材料体系、不同表面状态和不同产品系列。

交叉型与嵌套型选择：

| 研究类型 | 适用条件 | PCB/载板例子 |
| --- | --- | --- |
| 交叉型GRR | 每个评价人可重复测量同一件同一位置，测量不会改变样件状态 | 板厚、阻抗coupon、线宽、部分尺寸测量 |
| 嵌套型GRR | 样件不可被不同评价人重复测同一状态，或破坏性制样后样本差异不可忽略 | 微切片、剥离强度、热应力后截面、部分化学分析 |
| 稳定性研究 | 同一标准件或稳定样件可长期重复测量 | XRF标准片、阻抗标准件、厚度块、AOI金板 |
| 相关性研究 | 新旧设备、客户/供应商、内部/外部实验室需要比较结果 | 新TDR对旧TDR、厂内XRF对客户XRF、不同工厂切片结果比对 |

## 接收准则与风险分层

通用阈值只能作为训练口径，不能机械替代客户CSR和工程风险判断。接收准则应在MSA计划阶段写清楚，并与控制计划、PPAP提交等级和特殊特性等级一致。

| 场景 | 接收关注点 | 判定倾向 |
| --- | --- | --- |
| 安全/法规/客户特殊特性 | 漏判风险、偏倚、稳定性、GRR、ndc均要严格控制 | 不宜仅按成本理由条件接受 |
| 产品放行 | 测量系统相对公差的变差、边界误判风险 | 优先看P/T、偏倚和边界样本判定 |
| 过程控制/SPC | 测量系统相对过程变差、ndc、控制图可判异能力 | 优先看%Study Variation、ndc和稳定性 |
| 分选/100%检验 | 漏判率、误判率、重复判定、设备漂移 | 要评估边界样和长期稳定性 |
| 客户争议/仲裁 | 偏倚、实验室比对、测量不确定度、标准件溯源 | 需明确主判实验室、方法版本和决策规则 |

对临界规格判定，建议在SOP或质量协议中明确决策规则：是否采用保护带、是否考虑测量不确定度、复测几次、复测均值还是单次值有效、客户与供应商数据不一致时谁的实验室为准。没有决策规则时，边界样品最容易演变为客诉争议。

## 计量型MSA

### 分辨率和灵敏度

先检查分辨率，再做GRR。分辨率不足时，GRR数据可能看似“整齐”，但实际无法支撑过程控制。

典型风险：

- 极差图中只有1-3个可读极差值。
- 超过四分之一的极差为0。
- 量具最小刻度大于公差或过程变差的1/10。
- 数字化设备输出被软件四舍五入或算法平滑，掩盖真实波动。

PCB/载板例子：

- 用低分辨率卡尺测薄板厚度，无法识别压合波动。
- AOI线宽算法分辨率不足，不能支持精细线路过程能力。
- 翘曲测量支撑点和读数精度不足，导致客户装配争议无法复现。

### 偏倚

偏倚是测量平均值与参考值的差。偏倚大说明系统性不准。

常见原因：

- 仪器未校准、校准周期不合适或校准范围未覆盖实际使用范围。
- 标准件磨损、参考值错误或溯源不清。
- 夹具、探针、基准面或测点选择造成固定偏差。
- 使用了不适合该特性的量具或方法。
- 环境温湿度、振动、清洁度或预热不足。
- 软件常数、补偿系数或测试程序错误。

PCB/载板例子：

- XRF标准片与实际镀层体系不匹配，导致ENIG/ENEPIG厚度系统偏差。
- TDR校准不当或探针接触不稳定，导致阻抗读数偏高/偏低。
- 微切片制样角度偏斜，导致孔铜厚度偏差。

### 线性

线性看的是量程内不同水平的偏倚是否一致。一个量具在低值区准确，不代表高值区也准确。

适用场景：

- 测量范围宽，例如不同板厚、不同铜厚、不同阻抗、不同镀层厚度。
- 同一设备用于多个产品系列或多个规格等级。
- 客户投诉集中在某一规格段。

文章正文给出的学习口径包括线性误差比例和相关性要求；实际项目应以AIAG方法、统计软件输出和客户CSR为准，关注斜率、截距、置信区间和实际工程影响。

### 稳定性

稳定性是测量系统随时间的漂移。它通常用同一标准件或稳定样件定期测量，并用Xbar-R或I-MR控制图判断。

不稳定的常见原因：

- 仪器老化、磨损、保养不良。
- 标准件损坏或污染。
- 温湿度、清洁度、振动、照明、电磁环境变化。
- 操作方法、夹持方式、归零方式发生变化。
- 设备预热不足或软件/程序版本变化。

PCB/载板例子：

- 阻抗测试仪电缆和探针磨损导致长期漂移。
- 显微镜照明和焦距变化影响线宽/缺陷判定。
- XRF窗口、准直器或标准片状态变化影响镀层厚度。

### GRR

GRR是重复性和再现性的联合估计，用来判断测量系统变差相对总变差或公差是否足够小。

常见AIAG训练口径：

| 指标 | 常见解释 |
| --- | --- |
| `%GRR <= 10%` | 通常可接受 |
| `10% < %GRR <= 30%` | 结合特性风险、成本、用途和客户要求条件接受 |
| `%GRR > 30%` | 通常不可接受，需要改善 |
| `ndc >= 5` | 通常认为可用于过程区分；越高越利于过程控制 |

选择比较基准时要看用途：

- 用于产品放行：重点看测量系统变差相对公差是否足够小。
- 用于过程控制/SPC：重点看测量系统变差相对过程变差是否足够小。
- 过程能力不足时，不应简单用较大的过程变差掩盖测量系统问题。

GRR失败时的典型改善路径：

`确认样本覆盖范围 -> 检查分辨率 -> 校准/维修/更换量具 -> 改善夹具和定位 -> 固化测点与方法 -> 培训评价人 -> 控制环境 -> 重新MSA`

### GRR结果判读

GRR报告不要只看一个“Pass/Fail”。应同时看变差来源、图形、样本覆盖和工程用途。

| 输出项 | 解释重点 | 常见风险 |
| --- | --- | --- |
| EV/重复性 | 固定条件下的内部波动 | 探针接触、夹具定位、焦距/照明、样件表面粗糙 |
| AV/再现性 | 评价人或条件变化带来的差异 | 人员手法、读数习惯、测点理解、班次差异 |
| Part-to-Part | 样件之间真实差异 | 样本范围过窄会放大%GRR；样本范围过宽会掩盖测量问题 |
| %Study Variation | GRR相对总研究变差的比例 | 更适合判断过程控制和SPC可用性 |
| %Tolerance/P-T | GRR相对公差带的比例 | 更适合判断产品放行和规格符合性 |
| %Contribution | 方差贡献比例 | 与%Study Variation不是同一口径，不能混用 |
| ndc | 可区分的过程类别数，常见计算为 `1.41 * PV / GRR` 后取整 | ndc低说明测量系统无法有效区分过程层级 |
| 人员-零件交互 | 某些评价人只在某些样本上偏差大 | 多见于手动测量、外观灰区和测点理解不同 |

常用分析方法：

- 平均值-极差法简单直观，适合培训和常规量具研究。
- ANOVA法可拆出评价人、零件和交互项，更适合复杂测量系统和人员差异明显的场景。
- 图形分析要和数字一起看，例如均值图、极差图、评价人-零件交互图、散点图和残差图。

当GRR数字不好时，先确认样本范围是否合理。过程样本范围太窄会让%Study Variation变差，但这不代表量具一定完全不可用；相反，故意挑选跨度过大的样本会让%GRR变好，却不能证明量产过程控制足够可靠。

### 偏倚、线性和稳定性判读

偏倚、线性和稳定性属于位置类误差，不能被GRR替代。

| 项目 | 判读重点 | 改善方向 |
| --- | --- | --- |
| 偏倚 | 平均测量值与参考值的差是否有统计和工程意义 | 校准、修正系数、标准件、测点和夹具 |
| 线性 | 不同测量范围内偏倚是否一致 | 分量程校准、限定使用范围、分产品族MSA |
| 稳定性 | 同一标准件随时间是否漂移或出现特殊原因 | 点检、预热、维护、环境控制、软件版本锁定 |

偏倚显著但稳定时，可以评估修正系数或补偿程序；偏倚不稳定时，不应只靠补偿解决。线性失败时，应确认设备是否超出适用范围，例如同一XRF程序同时覆盖极薄和极厚镀层、同一二次元测量不同反光材料边缘。

### 破坏性和准破坏性测量

破坏性测量不能简单套普通交叉GRR，因为同一零件被测后状态可能已经改变。常见做法：

- 使用同质样本分组：从同一批、同一位置、同一工艺窗口取样，尽量降低样本间差异。
- 使用嵌套设计：评价人测不同但可视为同质的样本，统计时把样本嵌套在评价人或批次中。
- 固定制样方法：切片角度、镶埋、研磨、抛光、蚀刻、拍照倍率和测点必须标准化。
- 做方法验证：对制样损伤、边缘识别、失效定义和判定边界做专项确认。
- 区分测试变差和样本变差：剥离强度、CAF、HAST等结果常同时包含材料/结构真实差异。

PCB/载板中的微切片、孔铜厚度、树脂凹陷、镀层界面、剥离强度和热应力后截面，都应把制样过程纳入测量系统，而不是只评价显微镜或图像软件。

## 属性型MSA

属性型MSA适用于合格/不合格、通过/不通过、缺陷等级、外观限度样判定等场景。PCB行业的人工外观检验、AOI复判、AVI判级、阻焊桥连、露铜、压伤、色差、微切片缺陷分类都应重视属性MSA。

### Kappa

Kappa用于衡量属性判定的一致程度，既可以比较评价人与评价人之间，也可以比较评价人与标准答案之间。

常用公式：

`Kappa = (Po - Pe) / (1 - Pe)`

其中：

- `Po`：实际一致比例。
- `Pe`：按随机机会预期的一致比例。

文章图片中的训练口径：`Kappa > 0.75` 通常表示一致性较好，`Kappa < 0.40` 表示一致性差。实际接收准则应以客户CSR、内部标准和缺陷风险为准。

### 有效性、漏判率和误判率

属性MSA不能只看评价人之间是否一致，还要看是否与标准答案一致。

| 指标 | 含义 | 质量风险 |
| --- | --- | --- |
| 有效性 | 正确判断次数 / 总判断次数 | 反映整体判定能力 |
| 漏判率 | 不合格被判为合格的比例 | 客户流出风险最高 |
| 误判率 | 合格被判为不合格的比例 | 造成报废、返工和效率损失 |

训练材料中给出的参考分级可作为学习线索：有效性高、漏判率低、误判率低才可接受；有效性不足或漏判率偏高时，应优先改善标准样、作业指导、培训和判定方法。

### 信号探测法

信号探测法适用于属性判定中存在灰区的场景。其思想是识别总是被接受、总是被拒收和判定不稳定的区域，并用灰区宽度估计测量系统变差相对公差或过程变差的风险。

PCB/载板例子：

- 外观缺陷接近限度样边界时，不同检验员判定不一致。
- AOI真假点复判在细线路、阻焊反光、铜面污染处出现灰区。
- 微切片孔壁空洞、树脂凹陷、镀层边界接近客户判定限度。

### 属性样本设计

属性MSA的样本比计算公式更关键。样本集应覆盖真实判定难点，而不是只包含明显好品和明显坏品。

| 样本类型 | 目的 | 示例 |
| --- | --- | --- |
| 明显合格 | 确认不会过度误判 | 正常阻焊、正常铜面、清晰无缺陷切片 |
| 明显不合格 | 确认不会漏放重大缺陷 | 开短路、露铜、严重压伤、孔壁空洞超限 |
| 临界合格 | 检查人员是否过度拒收 | 接近限度样但仍可接受的色差、凹陷、偏移 |
| 临界不合格 | 检查漏判风险 | 刚超过客户限度的阻焊桥连、镀层异常、孔壁缺陷 |
| 易混缺陷 | 检查分类能力 | 异物/压伤/划伤、假点/真缺陷、镍腐蚀/污染 |

属性样本最好建立标准答案，并由客户规范、限度样、资深评审或跨部门评审确认。若标准答案本身有争议，应先修订判定标准和限度样，再做评价人一致性。

### 属性结果判读

属性MSA至少要同时看三类结果：

- 评价人与标准答案的一致性：评价人是否判得对。
- 评价人之间一致性：不同人员是否判得一致。
- 评价人自身重复性：同一人多轮盲测是否前后一致。

常见补充指标：

| 指标 | 含义 | 用途 |
| --- | --- | --- |
| 灵敏度/检出率 | 不合格被正确判为不合格的比例 | 控制漏放和客户流出 |
| 特异度 | 合格被正确判为合格的比例 | 控制过检、报废和返工 |
| 阳性预测值 | 判为不合格的样本中真实不合格比例 | 评估复判工作量和误报 |
| 阴性预测值 | 判为合格的样本中真实合格比例 | 评估放行可信度 |
| 加权Kappa | 对有等级顺序的判定赋予不同错误权重 | 适用于外观等级、可焊性等级、缺陷严重度 |

当样本类别极不均衡时，Kappa可能被样本结构影响。PCB外观检验中不良率通常很低，若样本集几乎都是好品，评价人看似一致，仍可能无法证明对关键缺陷有足够检出能力。因此，属性MSA样本集应有意识地加入不合格和边界样。

### 自动化判定系统

AOI、AVI、X-ray自动判定、图像测量软件和AI缺陷分类模型也属于测量系统。上线或变更时应验证：

- 程序/模型版本、参数、阈值、光源、相机、镜头、治具和样本库是否受控。
- 对不同阻焊颜色、铜面粗糙度、线路密度、反光状态和板面污染是否稳健。
- 漏报、误报、复判效率和缺陷分类混淆矩阵是否满足风险要求。
- 软件升级、模型再训练、相机更换、光源老化后是否触发再验证。
- 自动判定与人工复判、客户判定之间是否有定期相关性确认。

## 变差来源与改善方向

| 变差来源 | 典型问题 | 改善方向 |
| --- | --- | --- |
| 人员 | 手法不一、培训不足、疲劳、视差、判定理解不同 | 培训认证、盲测、限度样、标准作业、交叉复核 |
| 设备 | 分辨率不足、磨损、校准不当、预热不足、程序错误 | 校准/维护、升级设备、锁定程序版本、设备预热和点检 |
| 方法 | 测点、夹持、归零、取样、记录方式不一致 | 固化SOP、图示测点、测量路线、防错表单 |
| 标准 | 标准件磨损、参考值不清、溯源链断裂 | 标准件管理、周期确认、溯源证书、Master Sample |
| 零件 | 表面粗糙、变形、批次差异、样本不代表过程 | 样本分层、覆盖过程范围、明确可重复测量位置 |
| 环境 | 温湿度、照明、振动、尘土、电磁干扰 | 实验室环境控制、测量前平衡、清洁和防振 |
| 软件 | 算法参数、补偿系数、版本变化、数据导出错误 | 软件版本管控、权限控制、验证记录、变更评估 |

## 相关性和实验室比对

当测量数据跨工厂、跨设备、跨供应商或跨客户使用时，单一工厂内部MSA还不够。应建立相关性或实验室比对机制。

常见场景：

- 同一客户项目在多工厂生产，关键尺寸、阻抗、镀层厚度和外观判定需要统一口径。
- 新设备替代旧设备，新程序替代旧程序，需要证明结果连续性。
- 供应商数据与客户来料检验数据不一致，需要确认偏倚和决策规则。
- 内部实验室和第三方实验室报告差异较大，需要确认样本、方法、环境和数据处理口径。

比对设计要点：

| 项目 | 要点 |
| --- | --- |
| 样本 | 使用稳定、可重复测量或可分样的样本；覆盖规格边界和常见产品族 |
| 方法 | 明确测点、夹具、前处理、环境、软件版本和数据处理规则 |
| 标准 | 优先使用可溯源标准件、客户认可限度样或双方确认的参考样 |
| 统计 | 比较平均差、相关性、斜率、截距、重复性和边界判定一致性 |
| 决策 | 约定偏差可接受范围、仲裁方法、主判实验室和纠正措施 |

只看相关系数不够。两个实验室结果高度相关，也可能存在固定偏倚，导致规格边界附近判定相反。

## PCB/载板常见MSA场景

| 特性/判定 | 测量系统 | MSA重点 | 常见风险 |
| --- | --- | --- | --- |
| 线宽线距 | 显微镜、二次元、AOI算法 | 分辨率、重复性、再现性、线性 | 焦距、照明、边缘算法、测点规则不一致 |
| mSAP/SAP精细线路 | 高倍显微镜、SEM、AOI/AVI、图像软件 | 分辨率、边缘识别、属性漏判、软件版本 | 铜面粗糙、侧蚀、底铜残留、阻焊反光导致边界不清 |
| 阻抗 | TDR、阻抗coupon、探针治具 | 偏倚、稳定性、GRR | 校准件、探针接触、coupon设计、线缆磨损 |
| 孔铜/填孔 | 微切片、图像测量、X-ray/CT | 破坏性MSA、重复性、方法一致性 | 制样角度、抛磨质量、测点选择、判定标准 |
| 激光微孔/盲孔 | 二次元、X-ray/CT、切片、AOI | 分辨率、破坏性MSA、缺陷分类一致性 | 孔径锥度、残胶、偏位、填孔凹陷和界面判定 |
| ENIG/ENEPIG镀层 | XRF、标准片、测试程序 | 偏倚、线性、稳定性 | 标准片体系不匹配、测点/准直器、镀层模型 |
| OSP/沉锡/沉银表面 | 化学分析、外观、可焊性测试 | 方法验证、稳定性、属性一致性 | 膜厚薄、氧化、污染、储存时间和前处理影响 |
| 板厚/介质厚度 | 千分尺、切片、CMM | 分辨率、GRR、线性 | 压合厚度分布、测点位置、夹持力 |
| ABF/BT build-up介质 | 切片、CMM、显微图像 | 破坏性MSA、线性、测点规则 | 介质厚度薄、局部铜密度差异、切片角度 |
| 翘曲 | 平台、CMM、翘曲测试仪 | 夹具、环境、重复性、再现性 | 支撑点、温湿度、板件应力释放、读数方法 |
| 载板共面性/回流翘曲 | Shadow moire、3D量测、热态翘曲仪 | 稳定性、夹具、温度曲线、重复性 | 热循环曲线、支撑方式、样本吸湿、基板残余应力 |
| 离子污染 | ROSE、离子色谱 | 方法验证、稳定性、重复性 | 溶液、时间、温度、清洁度、样本处理 |
| 外观判定 | 人工目检、AVI、限度样 | Kappa、有效性、漏判率、误判率 | 光照、角度、培训、限度样不充分 |
| AOI真假点 | AOI程序、复判人员 | 属性一致性、软件版本、人员差异 | 算法阈值、反光、阻焊颜色、细线路误报 |
| BGA/焊盘尺寸与开窗 | 二次元、AOI、CMM、图像软件 | GRR、边缘识别、相关性 | 阻焊边缘不清、焊盘反光、涨缩补偿和测点定义 |
| 可靠性测试 | 热应力、CAF、HAST、剥离强度 | 破坏性/嵌套设计、方法一致性 | 样本同质性、试验条件、失效判定边界 |

## 测量系统优先级矩阵

不是所有测量项目都要用同样深度的MSA，但风险越高，越不能只靠经验判断。

| 优先级 | 典型场景 | 最低要求 | 复验触发 |
| --- | --- | --- | --- |
| A级 | 客户特殊特性、产品放行、PPAP主证据、边界分选、客户争议仲裁 | 分辨率、偏倚、稳定性、GRR或属性一致性、边界样本验证和明确决策规则 | 设备维修/搬迁、程序改版、客户判定争议、标准件变化 |
| B级 | SPC、过程能力、工艺优化和内部质量预警 | 分辨率、GRR、ndc、稳定性，必要时做相关性 | 产品族切换、量程变化、换班组或换工装 |
| C级 | 工程诊断、开发试验、辅助手段 | 方法确认、基础校准、样本代表性和使用边界说明 | 方法改变、用途升级到放行或PPAP时 |

对破坏性检测或高成本检测，不应因为样本贵就省略方法验证。可通过coupon、配对样、嵌套设计、相关性研究和更清晰的决策规则来降低试验成本，但不能跳过风险判断。

## MSA输出与记录

一个可审计的MSA包应至少包括：

- MSA计划和测量系统清单。
- 控制计划中特性、测量方法、设备编号和MSA报告的对应关系。
- 样本选择依据：覆盖范围、批次、规格、特殊特性、是否盲测。
- 评价人、重复次数、测量顺序和原始数据。
- 分辨率、偏倚、线性、稳定性、GRR、ndc或Kappa等分析结果。
- 接收准则、判定结论和批准记录。
- 不合格时的遏制措施、纠正措施、责任人、完成日期和再验证。
- 相关SOP、检验规范、限度样、培训记录、软件版本和标准件溯源记录。

建议报告字段：

| 字段 | 内容 |
| --- | --- |
| 报告基本信息 | 项目、客户、产品族、特性、规格、特殊特性等级、报告编号和版本 |
| 测量系统描述 | 设备编号、夹具、标准件、软件版本、SOP编号、环境条件和校准状态 |
| 研究设计 | 数据类型、研究方法、样本数量、评价人、重复次数、随机/盲测规则 |
| 原始数据 | 每次读数或判定、测量顺序、异常备注、剔除数据理由 |
| 统计结果 | 分辨率、偏倚、线性、稳定性、GRR、ndc、Kappa、漏判率/误判率等 |
| 工程解释 | 结果能否支持放行、SPC、Cpk、PPAP、分选或客户争议判定 |
| 结论与措施 | 接收/条件接收/拒收、限制使用范围、改善措施、责任人和再验证日期 |
| 批准与追溯 | 制作、审核、批准、客户批准状态、关联控制计划和PPAP资料 |

## 失效反应计划

MSA不合格时，不应继续无条件使用该测量数据做放行、SPC、Cpk或PPAP证据。建议按风险执行：

1. 立即评估已放行批次、在制品和客户风险。
2. 使用已验证的替代测量系统或第三方/客户认可实验室复核。
3. 对关键特性加严检验或暂停相关判定，直到测量系统恢复可信。
4. 分析失效来源：分辨率、偏倚、稳定性、重复性、再现性、属性一致性。
5. 改善设备、夹具、标准、SOP、环境、软件或人员培训。
6. 重新执行MSA并更新控制计划、检验规范和培训记录。
7. 若影响PPAP、客户批准或历史交付，应按CSR要求通知客户或提交变更/偏差资料。

## 常见错误

- 把校准证书当作MSA证据。
- 控制计划里列了测量设备，但没有对应MSA计划和报告。
- 只做GRR，不做分辨率、偏倚、线性和稳定性评估。
- 使用实验室高手测量，而不是实际量产检验员和现场条件。
- 样本只选标准件或“漂亮样件”，没有覆盖真实过程范围。
- 评价人知道标准答案或测量顺序固定，导致盲测失效。
- 属性检验只培训，不做Kappa、有效性、漏判率和误判率验证。
- 破坏性测试仍套用普通交叉GRR，忽略样本不可重复的事实。
- GRR用公差通过后，就直接用于SPC，忽略过程变差和ndc不足。
- 为了让GRR好看，故意选择跨度很大的样本，却没有代表量产过程。
- 只看相关系数，不看固定偏倚和规格边界附近的判定一致性。
- 自动化检测程序改版后没有重新验证，导致历史数据和当前判定口径不一致。
- MSA报告没有和控制计划、SOP、设备清单、限度样、培训记录建立追溯。
- MSA失败后只重算数据，不改善设备、方法、环境和人员。

## 快速落地流程

`控制计划测量项清单 -> 数据类型分类 -> 校准/标准/分辨率确认 -> 样本和评价人设计 -> 盲测/随机测量 -> 统计分析 -> 接收判定 -> 改善和再验证 -> 更新Control Plan/SOP/培训/PPAP资料`

使用建议：

- 做APQP或PPAP时，先从 `26-apqp.md` 确认特殊特性和第四阶段验证要求，再用本页设计MSA。
- 做IATF 16949或客户审核时，从控制计划抽样追溯到MSA报告、原始数据、判定准则和改善记录。
- 做AOI/AVI过杀、漏检、FQC外观判定一致性和ET复测争议时，结合 `36-defect-scrap-mapping-aoi-et-fqc.md` 判断测量系统对mapping数据的影响。
- 做PCB/载板制程改善时，先确认测量系统可信，再解释SPC异常、Cpk变化、良率波动和客户争议。
