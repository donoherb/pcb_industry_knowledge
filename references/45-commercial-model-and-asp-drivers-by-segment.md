## 用途

- 公司研究：解释“ASP 上升为何毛利未改善/现金变差”。
- 项目可研：把技术路线映射到**可交付合格产出**与回本节奏。
- 尽调：识别“看起来高端”的产品段，是否被折旧、COPQ、条款与爬坡期吞噬收益。

## 先对齐三个口径（否则所有模型都会跑偏）

1. **计价单位**：面积 / panel / strip / unit / 颗（package）与换算假设。
2. **成本口径**：投入成本 vs **良品成本**（建议默认良品成本，包含 COPQ 与关键验证成本）。
3. **产出口径**：名义产能 vs 有效产能 vs **合格可交付产出**（见 `41-capacity-yield-ramp-and-bottlenecks.md`）。

## 商业模型的“通用骨架”（建议固定）

```text
收入 = 出货量（合格产出） × ASP
毛利 = 收入 - (材料 + 转换成本 + 折旧 + COPQ + 工程/验证 + 物流包装)
经营性现金流 ≈ 毛利 - 营运资金增加 - Capex相关现金支出（简化）
```

把 ASP 拆成“可解释的溢价因子”，把成本拆成“会随复杂度非线性上升的项”。

## ASP 的六类驱动（从“技术”到“商业”）

1. **材料溢价**：低损耗/高 Tg/低粗糙铜箔/ABF 等（但伴随锁料与MOQ风险）。
2. **复杂度溢价**：层数、阶数、线宽线距、孔型、混压/背钻等（同时推高良率风险）。
3. **交付溢价**：急单、小批量、多变更（往往以“占用瓶颈产能”的形式体现在成本端）。
4. **认证溢价**：车规/服务器/封装厂认证（常伴随验证成本与导入期折旧压力）。
5. **服务溢价**：工程支持、联合开发、FA与驻厂（可能被低估为期间费用）。
6. **供需溢价**：景气/缺货周期（高度动态，必须标注日期与来源）。

## 成本的“非线性杀手”（利润常被这几项吃掉）

- **良率与返工**：复杂度提升 → 不是线性多花一点材料，而是指数放大 COPQ。
- **瓶颈占用**：AOI复判、实验室吞吐、首件确认与换线时间，常比设备数量更致命。
- **折旧压力**：新线爬坡期最重，且与认证节奏强相关。
- **条款风险**：降价、返利、扣款、索赔、账期（见 `40-quotation-costing-and-commercial-terms.md`）。
- **营运资金**：锁料、在制周期、验证批与版本变更导致的呆滞与占用。

## 分产品段：如何把“技术参数”映射到“经营结果”

### A) 标准多层/中低端刚性板

- **ASP 驱动**：交付与规模为主，技术溢价有限。
- **关键经营变量**：稼动率、换线效率、材料采购与损耗、OTD 与库存周转。
- **常见误判**：只看收入增长，忽略价格竞争导致的条款恶化与COPQ上升。

### B) 高多层 / 大尺寸（服务器背板/通信部分场景）

- **ASP 驱动**：层数、尺寸、阻抗控制与可靠性交付能力。
- **成本敏感项**：压合/钻孔/电镀与终检吞吐；返工成本很高。
- **尽调要点**：瓶颈工序与实验室是否跟得上；阻抗争议与相关性能力。

### C) 高速/高频板（交换机/服务器/通信/雷达）

- **ASP 驱动**：材料体系（Dk/Df、低粗糙）、SI/PI一致性、阻抗与损耗指标、认证门槛。
- **成本敏感项**：材料锁定与MOQ、报废与复测、阻抗相关性维护、变更纪律。
- **现金流风险**：验证周期长、在制周期长、客户账期与扣款敏感。

### D) HDI / 任意层 / 类载板工艺

- **ASP 驱动**：互连结构（叠孔/填孔）、细线能力、稳定良率与交付。
- **成本敏感项**：填孔/对位/细线缺陷导致 COPQ；AOI过杀与复判吞吐；工程释放与首件确认。
- **常见误判**：把“样品能做”当成“量产能赚钱”；忽略爬坡期折旧与COPQ。

### E) FPC / 软硬结合

- **ASP 驱动**：机械可靠性（弯折寿命）、材料体系与贴装协同。
- **成本敏感项**：报废与返工、贴装良率外溢、来料一致性与储存窗口。
- **尽调要点**：动态/静态弯折验证与失效机理闭环；与客户装配窗口一致性。

### F) IC载板（BT/ABF，FC-CSP/FC-BGA）

- **ASP 驱动**：线宽线距/对位/洁净度、翘曲与尺寸稳定、封装协同、认证与供货资格。
- **成本敏感项**：良率窗口、返工几乎不可承受；材料（ABF）锁定与供给风险；验证与客户放行节奏。
- **关键风险**：收入先到但利润/现金后到；扩产与导入延期会把折旧与现金流放大成系统性风险。

## 快速“投资/可研”判断：三句话框架

1) ASP 上升是靠**材料/复杂度/认证**还是靠**供需周期**？  
2) 这条路线的盈利决定项是**良率/瓶颈/折旧/COPQ/条款**中的哪两项？证据是什么？  
3) 若导入延期一个季度，现金流是否仍可承受（营运资金 + 折旧）？
